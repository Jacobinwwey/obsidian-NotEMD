## त्वरित शुरुआत

1.  **इंस्टॉल और सक्षम करें**: Obsidian कम्युनिटी प्लगइन स्टोर से प्लगइन प्राप्त करें।
2.  **LLM कॉन्फ़िगर करें**: `Settings -> Notemd` पर जाएं, अपना LLM प्रदाता (जैसे OpenAI या स्थानीय प्रदाता जैसे Ollama) चुनें, और अपनी API कुंजी/URL दर्ज करें।
3.  **साइडबार खोलें**: साइडबार खोलने के लिए बाएं रिबन में Notemd वैंड आइकन पर क्लिक करें।
4.  **नोट संसाधित करें**: कोई भी नोट खोलें और मुख्य अवधारणाओं में `[[wiki-links]]` स्वचालित रूप से जोड़ने के लिए साइडबार में **"Process File (Add Links)"** पर क्लिक करें।
5.  **त्वरित वर्कफ़्लो चलाएँ**: प्रोसेसिंग, बैच जनरेशन और Mermaid सुधार को एक ही स्थान से जोड़ने के लिए डिफ़ॉल्ट **"One-Click Extract"** बटन का उपयोग करें।

बस! वेब रिसर्च, अनुवाद और सामग्री निर्माण जैसी अधिक सुविधाओं को अनलॉक करने के लिए सेटिंग्स देखें।

## भाषा समर्थन

### भाषा व्यवहार अनुबंध

| पहलू | कार्यक्षेत्र | डिफ़ॉल्ट | नोट्स |
|---|---|---|---|
| `UI Locale` | केवल प्लगइन इंटरफ़ेस टेक्स्ट | `auto` | Obsidian लोकेल का अनुसरण करता है; वर्तमान में `en`, `zh-CN`, `zh-TW` समर्थित हैं। |
| `कार्य आउटपुट भाषा` | LLM द्वारा उत्पन्न आउटपुट (लिंक, सारांश) | `en` | वैश्विक या प्रति-कार्य हो सकता है। |
| `ऑटो अनुवाद अक्षम करें` | गैर-अनुवाद कार्य मूल संदर्भ रखते हैं | `false` | स्पष्ट `Translate` कार्य अभी भी लक्ष्य भाषा लागू करते हैं। |

- आधिकारिक दस्तावेज़ अंग्रेजी और सरलीकृत चीनी में बनाए रखे जाते हैं, जिसमें 30 से अधिक भाषाओं के लिए पूर्ण समर्थन है।
- सभी समर्थित भाषाएं ऊपर हेडर में लिंक की गई हैं।

## मुख्य विशेषताएं

### AI-संचालित दस्तावेज़ प्रसंस्करण
- **बहु-LLM समर्थन**: विभिन्न क्लाउड और स्थानीय LLM प्रदाताओं से जुड़ें।
- **स्मार्ट चंकिंग**: बड़े दस्तावेजों को स्वचालित रूप से प्रबंधनीय टुकड़ों में विभाजित करता है।
- **सामग्री संरक्षण**: संरचना और लिंक जोड़ते समय मूल स्वरूपण बनाए रखता है।
- **पुनः प्रयास तर्क**: विफल API कॉल के लिए वैकल्पिक स्वचालित पुनः प्रयास।
- **चीन-तैयार प्रीसेट**: `Qwen`, `Doubao`, `Moonshot` आदि प्रदाता शामिल हैं।

### ज्ञान ग्राफ़ संवर्धन
- **स्वचालित विकी-लिंकिंग**: मुख्य अवधारणाओं की पहचान करता है और विकी-लिंक जोड़ता है।
- **अवधारणा नोट निर्माण**: खोजी गई अवधारणाओं के लिए स्वचालित रूप से नए नोट बनाता है।

### अनुवाद
- **AI-संचालित अनुवाद**: कॉन्फ़िगर किए गए LLM का उपयोग करके नोट सामग्री का अनुवाद करें।
- **बैच अनुवाद**: चयनित फ़ोल्डर के भीतर सभी फ़ाइलों का अनुवाद करें।

### वेब अनुसंधान और सामग्री निर्माण
- **वेब रिसर्च**: Tavily या DuckDuckGo का उपयोग करके खोजें और सारांशित करें।
- **शीर्षक से सामग्री निर्माण**: नोट शीर्षक का उपयोग करके प्रारंभिक सामग्री उत्पन्न करें।
- **Mermaid ऑटो-फिक्स**: उत्पन्न Mermaid आरेखों के सिंटैक्स को स्वचालित रूप से ठीक करता है।

## स्थापना
1. Obsidian **Settings** → **Community plugins** खोलें।
2. "Restricted mode" बंद करना सुनिश्चित करें।
3. **Browse** पर क्लिक करें और "Notemd" खोजें।
4. **Install** पर क्लिक करें और फिर **Enable** करें।
