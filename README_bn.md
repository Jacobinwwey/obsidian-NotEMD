## দ্রুত শুরু

১.  **ইনস্টল এবং সক্রিয় করুন**: ওবিসিডিয়ান কমিউনিটি প্লাগইন স্টোর থেকে প্লাগইনটি নিন।
২.  **LLM কনফিগার করুন**: `Settings -> Notemd`-এ যান, আপনার LLM প্রদানকারী (যেমন OpenAI বা লোকাল প্রদানকারী যেমন Ollama) বেছে নিন এবং আপনার API কী/URL প্রদান করুন।
৩.  **সাইডবার খুলুন**: সাইডবার খোলার জন্য বাম পাশের টুলবারের Notemd জাদুদণ্ড আইকনে ক্লিক করুন।
৪.  **নোট প্রসেস করুন**: যেকোনো নোট খুলুন এবং সাইডবারে **"Process File (Add Links)"**-এ ক্লিক করুন যাতে স্বয়ংক্রিয়ভাবে `[[wiki-links]]` যুক্ত হয়।
৫.  **কুইক ওয়ার্কফ্লো চালান**: এক ক্লিকেই প্রসেসিং, ব্যাচ জেনারেশন এবং Mermaid ফিক্স করার জন্য ডিফল্ট **"One-Click Extract"** বাটনটি ব্যবহার করুন।

ব্যাস! ওয়েব রিসার্চ, অনুবাদ এবং কন্টেন্ট জেনারেশনের মতো আরও ফিচার ব্যবহার করতে সেটিংস দেখুন।

## ভাষা সমর্থন

### ভাষা ব্যবহারের নিয়মাবলী

| বিষয় | পরিধি | ডিফল্ট | নোট |
|---|---|---|---|
| `UI Locale` | শুধুমাত্র প্লাগইন ইন্টারফেস টেক্সট | `auto` | ওবিসিডিয়ান ভাষা অনুসরণ করে; বর্তমানে `en`, `zh-CN`, `zh-TW` সমর্থিত। |
| `Task Output Language` | এআই দ্বারা তৈরি ফলাফল (লিঙ্ক, সারসংক্ষেপ) | `en` | সবকিছুর জন্য এক বা প্রতিটি কাজের জন্য আলাদা ভাষা হতে পারে। |
| `Disable auto translation` | অনুবাদ ছাড়া অন্য কাজে মূল ভাষা বজায় রাখা | `false` | নির্দিষ্টভাবে `Translate` কাজে লক্ষ্য ভাষা ব্যবহার করা হবে। |

- অফিশিয়াল ডকুমেন্টেশন ইংরেজি এবং সরলীকৃত চীনা ভাষায় রাখা হয়, সাথে ৩০টিরও বেশি ভাষার পূর্ণ সমর্থন রয়েছে।
- সমর্থিত সব ভাষার লিঙ্ক উপরের হেডারে দেওয়া আছে।

## মূল বৈশিষ্ট্যসমূহ

### এআই-চালিত ডকুমেন্ট প্রসেসিং
- **মাল্টি-LLM সমর্থন**: বিভিন্ন ক্লাউড এবং লোকাল LLM প্রদানকারীর সাথে সংযোগ।
- **স্মার্ট চাঙ্কিং**: বড় ডকুমেন্টগুলোকে স্বয়ংক্রিয়ভাবে ছোট ছোট অংশে ভাগ করে প্রসেস করা।
- **ফরম্যাট বজায় রাখা**: লিঙ্ক যুক্ত করার সময় মূল ফরম্যাট ঠিক রাখার চেষ্টা করা।
- **রিট্রাই লজিক**: API কল ব্যর্থ হলে স্বয়ংক্রিয়ভাবে পুনরায় চেষ্টা করার সুবিধা।
- **চীন-প্রস্তুত প্রিসেট**: `Qwen`, `Doubao`, `Moonshot` ইত্যাদি প্রদানকারী অন্তর্ভুক্ত।

### নলেজ গ্রাফ উন্নত করা
- **স্বয়ংক্রিয় উইকি-লিঙ্কিং**: মূল ধারণাগুলো চিনে `[[লিঙ্ক]]` যুক্ত করা।
- **কনসেপ্ট নোট তৈরি**: নতুন ধারণাগুলোর জন্য স্বয়ংক্রিয়ভাবে নতুন নোট তৈরি করা।

### অনুবাদ
- **এআই অনুবাদ**: পছন্দমতো ভাষায় নোটের কন্টেন্ট অনুবাদ করা।
- **ব্যাচ অনুবাদ**: একটি ফোল্ডারের সব ফাইল একসাথে অনুবাদ করা।

### ওয়েব রিসার্চ এবং কন্টেন্ট তৈরি
- **ওয়েব রিসার্চ**: Tavily বা DuckDuckGo ব্যবহার করে তথ্য খোঁজা এবং সারসংক্ষেপ তৈরি।
- **শিরোনাম থেকে কন্টেন্ট তৈরি**: নোটের শিরোনাম ব্যবহার করে এআই দিয়ে কন্টেন্ট তৈরি।
- **Mermaid অটো-ফিক্স**: তৈরি করা Mermaid ডায়াগ্রামের ভুল স্বয়ংক্রিয়ভাবে ঠিক করা।

## ইনস্টলেশন
১. ওবিসিডিয়ান **Settings** → **Community plugins**-এ যান।
২. "Restricted mode" বন্ধ আছে কিনা নিশ্চিত করুন।
৩. **Browse**-এ ক্লিক করে "Notemd" লিখে খুঁজুন।
৪. **Install**-এ ক্লিক করুন এবং তারপর **Enable** করুন।
