## 翻译状态策略

- 权威源文档：仓库中的 English + 简体中文。
- 机器辅助链接是 best-effort，可能存在时效滞后。
- 如机器翻译与源文档冲突，以源文档为准。

## 贡献人工翻译

1. 在 `docs/i18n/` 下新增语言文件，例如：`docs/i18n/README.es.md`。
2. 章节顺序尽量与 `README.md` 保持一致。
3. 在文档顶部标注状态（`Draft` 或 `Reviewed`）。
4. 提交 PR，条件允许时至少邀请一位母语贡献者复核。
