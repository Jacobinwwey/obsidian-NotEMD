# Notemd v1.8.3

## Highlights

- **Welcome Modal**: Shown on first install with plugin capabilities overview, LLM setup hint, and sponsor buttons. Fully i18n (22 locales).
- **Sponsor Support**: Settings tab sponsor section with GitHub Star + ko-fi Coffee buttons. manifest.json fundingUrl for Obsidian community browser.
- **Cline-Aligned Token Resolution**: Unknown models with default maxTokens now defer to the API provider instead of being capped at 8192.
- **Diagram Edge Normalization**: LLM outputs using `source`/`target` edge fields normalized to `from`/`to`.
