### 核心亮点
这个补丁版本补上了 `1.8.0` 之后仍然存在的语言支持缺口。Notemd 现已完成所有已宣称支持的应用内 UI locale 的可见界面本地化收口，包括设置页、侧边栏动作、工作流构建器控件、原文提取设置、高级选项以及输出路径配置等界面，都应跟随 Obsidian 当前语言，而不是在正常使用中静默回退到英文。

同时，这一版也修正了文档契约。仓库根 README 与语言中心现在明确区分维护源文档、已发布 README 译文以及应用内 UI locale 的精确目录。English fallback 仍保留在实现中作为安全网，但不再被文档表述为受支持 locale 的常态路径。

### 破坏性更新
* **无需用户侧迁移操作**：现有设置可继续使用，`uiLocale: auto` 仍然跟随 Obsidian。

### 新特性
* `feat`: 补齐所有受支持 UI locale 在剩余设置项、工作流构建器、侧边栏提示、原文提取、高级设置与输出路径配置等可见界面上的本地化覆盖。
* `feat`: 新增工作流构建器、侧边栏提示、原文提取设置、剩余高级设置与通用输出路径界面的定向回归测试。
* `feat`: 更新语言中心，公开仓库中已发布的 README 译文集合，以及代码中精确的应用内 UI locale 目录。

### Bug 修复
* `fix`: 修复了安装或更新后，部分已支持 locale 仍在某些设置项和侧边栏界面显示英文的问题，根因是这些可见界面仍存在遗漏的本地化项。
* `fix`: 修复了 README 与语言中心仍以“30+ 语言”笼统表述、未区分文档译文与应用内 UI locale 覆盖面的文档漂移问题。
* `fix`: 修复了语言支持说明中的契约偏差，明确 English fallback 是实现层安全网，而不是受支持可见界面的默认表现。

### 重构与杂项
* `docs`: 按当前代码现状对齐 `README.md`、`README_zh.md`、各语种 README 与语言中心。
* `chore`: 同步 `package.json`、`manifest.json`、`versions.json`、`change.md` 与 `1.8.1` 的发布元数据。
