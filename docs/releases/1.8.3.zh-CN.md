# Notemd v1.8.3

## 亮点

- **欢迎弹窗**：首次安装时展示，包含插件能力概述、LLM 配置提示和赞助按钮。22 种语言完整 i18n。
- **赞助支持**：设置页赞助板块，含 GitHub 加星 + ko-fi 咖啡按钮。manifest.json fundingUrl 支持 Obsidian 社区浏览器展示。
- **Cline 对齐令牌解析**：未知模型使用默认 maxTokens 时由 API 提供商自行决定，不再限制在 8192。
- **图表边缘规范化**：LLM 输出使用 `source`/`target` 边缘字段现被规范化为 `from`/`to`。
