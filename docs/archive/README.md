# Documentation Archive

Language: **English** | [简体中文](./README.zh-CN.md)

This directory keeps historical or one-off documentation that is still worth preserving but should no longer be treated as current repository truth.

## Archive Areas

- [`root-history/`](./root-history/) - historical Markdown files that previously lived in the repository root and made the root path noisy

## How To Use The Archive

- Use archive docs for historical context, migration notes, or forensic reference.
- Do not treat archive docs as the default source of truth when a current maintainer or progress doc already exists under `docs/`.
- When in doubt, start from [`../README.md`](../README.md) or [`../maintainer/repository-document-layout.md`](../maintainer/repository-document-layout.md).
