Простой способ создать свою базу знаний!

Notemd расширяет ваш рабочий процесс в Obsidian, интегрируясь с различными большими языковыми моделями (LLM). Он поддерживает многоязычную обработку заметок, автоматически создает вики-ссылки для ключевых понятий, создает соответствующие концептуальные заметки, выполняет веб-поиск и суммаризацию, перевод и визуализацию в виде диаграмм Mermaid, помогая строить мощный граф знаний.

## Быстрый старт

1.  **Установка и включение**: Установите плагин из магазина сообщества Obsidian.
2.  **Настройка LLM**: Перейдите в `Настройки -> Notemd`, выберите провайдера LLM (например, OpenAI или локальный Ollama) и введите API ключ/URL.
3.  **Откройте боковую панель**: Нажмите на иконку волшебной палочки Notemd на левой панели.
4.  **Обработка заметки**: Откройте заметку и нажмите **«Обработать файл (добавить ссылки)»** в боковой панели.
5.  **Запуск быстрого процесса**: Используйте кнопку **«One-Click Extract»** для последовательного выполнения обработки, генерации и исправления Mermaid.

Готово! Изучите настройки для доступа к веб-поиску, переводу и генерации контента.

## Языковая поддержка

### Языковой контракт

| Аспект | Область | По умолчанию | Примечание |
|---|---|---|---|
| `UI Locale` | Только текст интерфейса плагина | `auto` | Согласно языку Obsidian; доступны `en`, `zh-CN`, `zh-TW`. |
| `Язык вывода` | Вывод задач LLM (ссылки, резюме) | `en` | Можно задать глобально или для каждой задачи. |
| `Отключить автоперевод` | Сохранение контекста оригинала | `false` | Задачи перевода всё равно используют целевой язык. |

- Официальная документация ведется на английском и упрощенном китайском, поддерживается более 30 языков.
- Все языки доступны по ссылкам в заголовке.

## Основные возможности

### Обработка документов с помощью ИИ
- **Поддержка множества LLM**: Облачные и локальные провайдеры.
- **Умное разбиение**: Автоматическое деление больших файлов на части.
- **Сохранение структуры**: Добавление ссылок без изменения исходного форматирования.
- **Логика повторов**: Автоматические повторные запросы при сбоях API.
- **Пресеты для Китая**: Включая `Qwen`, `Doubao`, `Moonshot` и др.

### Расширение графа знаний
- **Авто-вики-ссылки**: Идентификация понятий и добавление `[[ссылок]]`.
- **Создание концепт-заметок**: Автоматическое создание новых файлов для обнаруженных понятий.

### Перевод
- **ИИ-перевод**: Перевод контента с помощью выбранной LLM.
- **Пакетный перевод**: Перевод всех файлов в выбранной папке.

### Веб-исследование и генерация
- **Веб-поиск и резюме**: Поиск через Tavily или DuckDuckGo.
- **Генерация по заголовку**: Создание контента на основе названия заметки.
- **Авто-исправление Mermaid**: Исправление синтаксиса созданных диаграмм.

## Установка
1. Откройте Obsidian **Настройки** → **Сторонние плагины**.
2. Отключите «Безопасный режим».
3. Нажмите **Обзор** и найдите «Notemd».
4. Нажмите **Установить**, затем **Включить**.
