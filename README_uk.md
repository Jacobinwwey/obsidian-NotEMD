Простий спосіб створити власну базу знань!

Notemd покращує ваш робочий процес в Obsidian, інтегруючись з різними великими мовними моделями (LLM) для обробки ваших багатомовних нотаток, автоматичного створення вікі-посилань для ключових понять, створення відповідних концептуальних нотаток, виконання веб-пошуку та допомоги в побудові потужних графів знань.

## Швидкий старт

1.  **Встановлення та активація**: Отримайте плагін з магазину спільноти Obsidian.
2.  **Налаштування LLM**: Перейдіть до `Налаштування -> Notemd`, виберіть провайдера LLM (наприклад, OpenAI або локальний Ollama) і введіть API ключ/URL.
3.  **Відкрийте бічну панель**: Натисніть на іконку чарівної палички Notemd на лівій панелі.
4.  **Обробка нотатки**: Відкрийте нотатку і натисніть **«Обробити файл (додати посилання)»** на бічній панелі для автоматичного додавання `[[wiki-links]]`.
5.  **Запуск швидкого процесу**: Використовуйте кнопку **«One-Click Extract»** для послідовного виконання обробки, генерації та виправлення Mermaid.

Готово! Вивчіть налаштування для доступу до веб-пошуку, перекладу та генерації контенту.

## Мовна підтримка

### Мовний контракт

| Аспект | Область | Типово | Примітка |
|---|---|---|---|
| `UI Locale` | Лише текст інтерфейсу плагіна | `auto` | Відповідно до мови Obsidian; доступні `en`, `zh-CN`, `zh-TW`. |
| `Мова виводу` | Вивід завдань LLM (посилання, резюме) | `en` | Можна задати глобально або для кожного завдання. |
| `Вимкнути автопереклад` | Завдання без перекладу зберігають оригінал | `false` | Завдання перекладу все одно використовують цільову мову. |

- Офіційна документація ведеться англійською та спрощеною китайською мовами, підтримується понад 30 мов.
- Всі мови доступні за посиланнями в заголовку.

## Основні можливості

### Обробка документів за допомогою ШІ
- **Підтримка багатьох LLM**: Хмарні та локальні провайдери.
- **Розумне розбиття**: Автоматичне ділення великих файлів на частини.
- **Збереження структури**: Додавання посилань без зміни вихідного форматування.
- **Логіка повторів**: Автоматичні повторні запити при збоях API.
- **Пресет для Китаю**: Включаючи `Qwen`, `Doubao`, `Moonshot` та ін.

### Розширення графа знань
- **Авто-вікі-посилання**: Ідентифікація понять та додавання `[[посилань]]`.
- **Створення концепт-нотаток**: Автоматичне створення нових файлів для виявлених понять.

### Переклад
- **ШІ-переклад**: Переклад контенту за допомогою обраної LLM.
- **Пакетний переклад**: Переклад усіх файлів у вибраній папці.

### Веб-дослідження та генерація
- **Веб-пошук та резюме**: Пошук через Tavily або DuckDuckGo.
- **Генерація за заголовком**: Створення контенту на основі назви нотатки.
- **Авто-виправлення Mermaid**: Виправлення синтаксису створених діаграм.

## Встановлення
1. Відкрийте Obsidian **Налаштування** → **Сторонні плагіни**.
2. Вимкніть «Безпечний режим».
3. Натисніть **Огляд** і знайдіть «Notemd».
4. Натисніть **Встановити**, потім **Увімкнути**.
