วิธีง่ายๆ ในการสร้างฐานความรู้ของคุณเอง!

Notemd ช่วยเพิ่มประสิทธิภาพการทำงานใน Obsidian โดยการรวมเข้ากับโมเดลภาษาขนาดใหญ่ (LLMs) ต่างๆ เพื่อประมวลผลบันทึกหลายภาษาของคุณ สร้างลิงก์วิกิสำหรับแนวคิดหลักโดยอัตโนมัติ สร้างบันทึกแนวคิดที่เกี่ยวข้อง ทำการค้นคว้าข้อมูลบนเว็บ และช่วยคุณสร้างกราฟความรู้ที่ทรงพลัง

## เริ่มต้นใช้งานด่วน

1.  **ติดตั้งและเปิดใช้งาน**: รับปลั๊กอินจากร้านค้าชุมชน Obsidian
2.  **กำหนดค่า LLM**: ไปที่ `Settings -> Notemd` เลือกผู้ให้บริการ LLM ของคุณ (เช่น OpenAI หรือผู้ให้บริการในเครื่องอย่าง Ollama) และป้อน API key/URL
3.  **เปิดแถบข้าง**: คลิกไอคอนไม้กายสิทธิ์ Notemd ที่แถบเครื่องมือด้านซ้ายเพื่อเปิดแถบข้าง
4.  **ประมวลผลบันทึก**: เปิดบันทึกใดก็ได้แล้วคลิก **"Process File (Add Links)"** ในแถบข้างเพื่อเพิ่ม `[[wiki-links]]` ให้กับแนวคิดหลักโดยอัตโนมัติ
5.  **รันเวิร์กโฟลว์ด่วน**: ใช้ปุ่มเริ่มต้น **"One-Click Extract"** เพื่อเชื่อมต่อการประมวลผล การสร้างแบบกลุ่ม และการซ่อมแซม Mermaid จากจุดเดียว

เสร็จเรียบร้อย! สำรวจการตั้งค่าเพื่อปลดล็อกคุณสมบัติเพิ่มเติม เช่น การค้นคว้าเว็บ การแปลภาษา และการสร้างเนื้อหา

## การสนับสนุนภาษา

### ข้อตกลงพฤติกรรมทางภาษา

| ด้าน | ขอบเขต | ค่าเริ่มต้น | หมายเหตุ |
|---|---|---|---|
| `UI Locale` | เฉพาะข้อความอินเทอร์เฟซปลั๊กอิน | `auto` | ตามภาษาของ Obsidian; ปัจจุบันรองรับ `en`, `zh-CN`, `zh-TW` |
| `ภาษาที่ส่งออกงาน` | ผลลัพธ์ที่สร้างโดย LLM (ลิงก์, บทสรุป) | `en` | ตั้งค่าได้ทั้งแบบส่วนกลางหรือแยกตามงาน |
| `ปิดการแปลอัตโนมัติ` | งานที่ไม่ใช่การแปลจะคงบริบทเดิมไว้ | `false` | งาน `Translate` ที่ชัดเจนจะยังคงใช้ภาษาเป้าหมาย |

- เอกสารอย่างเป็นทางการได้รับการดูแลในภาษาอังกฤษและภาษาจีนตัวย่อ โดยรองรับมากกว่า 30 ภาษาอย่างสมบูรณ์
- ลิงก์ภาษาที่รองรับทั้งหมดอยู่ในหัวข้อด้านบน

## คุณสมบัติหลัก

### การประมวลผลเอกสารด้วย AI
- **รองรับหลาย LLM**: เชื่อมต่อกับผู้ให้บริการ LLM ทั้งบนคลาวด์และในเครื่อง
- **การแบ่งส่วนอัจฉริยะ**: แบ่งเอกสารขนาดใหญ่เป็นส่วนๆ โดยอัตโนมัติเพื่อให้จัดการได้ง่าย
- **การรักษาเนื้อหาเดิม**: พยายามรักษารูปแบบเดิมไว้ในขณะที่เพิ่มโครงสร้างและลิงก์
- **ตรรกะการลองใหม่**: ตัวเลือกการลองใหม่อัตโนมัติเมื่อการเรียก API ล้มเหลว
- **ค่าล่วงหน้าสำหรับจีน**: รวมถึงผู้ให้บริการเช่น `Qwen`, `Doubao`, `Moonshot` เป็นต้น

### การเพิ่มประสิทธิภาพกราฟความรู้
- **ลิงก์วิกิอัตโนมัติ**: ระบุและเพิ่มลิงก์วิกิไปยังแนวคิดหลัก
- **การสร้างบันทึกแนวคิด**: สร้างบันทึกใหม่โดยอัตโนมัติสำหรับแนวคิดที่ค้นพบ

### การแปลภาษา
- **การแปลด้วย AI**: แปลเนื้อหาบันทึกโดยใช้ LLM ที่กำหนดค่าไว้
- **การแปลแบบกลุ่ม**: แปลไฟล์ทั้งหมดในโฟลเดอร์ที่เลือก

### การค้นคว้าเว็บและการสร้างเนื้อหา
- **การค้นคว้าเว็บ**: ค้นหาผ่าน Tavily หรือ DuckDuckGo และสรุปผลลัพธ์
- **สร้างจากชื่อเรื่อง**: ใช้ชื่อเรื่องบันทึกเพื่อสร้างเนื้อหาเริ่มต้น
- **ซ่อมแซม Mermaid อัตโนมัติ**: ซ่อมแซมไวยากรณ์ของแผนภาพ Mermaid ที่สร้างขึ้นโดยอัตโนมัติ

## การติดตั้ง
1. เปิด Obsidian **Settings** → **Community plugins**
2. ตรวจสอบให้แนใจว่าปิด "Restricted mode"
3. คลิก **Browse** และค้นหา "Notemd"
4. คลิก **Install** แล้วคลิก **Enable**
