# OpenAI Provider

<TLDR>
**OpenAI is configured as part of the [LLM Providers](/docs/providers/overview) overview.** Key details: default model `gpt-4o`, base URL `api.openai.com/v1`, temperature `0.5`. Supports model discovery via `/v1/models` endpoint. Requires an API key with sufficient credits.

This is part of the [Obsidian AI Knowledge Management Guide](/docs/pillar-ai-knowledge).
</TLDR>

Full setup instructions, per-task configuration, and troubleshooting are covered in [LLM Providers](/docs/providers/overview).

**Quick reference:**

| Field | Value |
|-------|-------|
| Default model | `gpt-4o` |
| Base URL | `api.openai.com/v1` |
| Temperature | `0.5` |
| Model discovery | `/v1/models` |
| API key | Required (pay-as-you-go) |
