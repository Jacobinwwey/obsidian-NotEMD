# Diagrame

<TLDR>
Alegeți **tipul de diagramă de care aveți nevoie**, apoi alegeți **formatul sursă editabil** al acesteia. Previsualizați rezultatul în Obsidian și exportați-l ca SVG, PNG sau PDF.
</TLDR>

## Înțelegeți cele trei opțiuni

| Opțiune | Ce controlează | Exemple |
|---|---|---|
| Tip diagramă | Semnificația diagramării | Fluxogramă, diagramă de secvență, diagramă electrică, grafic de date |
| Format sursă | Fișierul editabil sau reutilizabil creat de Notemd | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| Format export | Imaginea sau documentul utilizat pentru partajare | SVG, PNG, PDF |

Draw.io și Drawnix sunt **formate sursă**, nu tipuri de diagrame. CircuitikZ este un format sursă conceput special pentru diagrame electrice. Aceeași previsualizare poate fi exportată în mai multe formate vizuale, deci alegerea CircuitikZ nu vă limitează doar la SVG.

## Alegeți un format sursă

| Format | Cel mai potrivit pentru | Fișier sursă |
|---|---|---|
| Mermaid | Diagrame păstrate direct în Markdown | Bloc de cod Mermaid |
| JSON Canvas | Fluxuri de lucru native în Obsidian | `.canvas` |
| Draw.io | Editare ulterioară în diagrams.net sau instrumente compatibile | `.drawio` |
| Drawnix | Editare în stil de desen manual sau pe tablou | `.drawnix` |
| CircuitikZ | Circuite electrice și de tranzistoare structurate | `.tex` |

Lăsați opțiunea **Format sursă preferat** pe **Auto** dacă nu aveți nevoie de un editor specific. Notemd alege un format compatibil pentru tipul de diagramă selectat.

## Generați și exportați o diagramă

1. Deschideți nota sau selectați textul pe care doriți să-l vizualizați.
2. Alegeți opțiunea **Genera diagramă** din bara laterală a Notemd sau din panoul de comenzi.
3. Selectați un **tip de diagramă**, sau lăsați-o pe **Detectare automată**.
4. Selectați un **format sursă**, sau lăsați-l pe **Auto**.
5. Verificați previsualizarea.
6. Alegeți opțiunea **Salva fișier sursă**, **Exporta SVG**, **Exporta PNG** sau **Exporta PDF**.

PNG și PDF folosesc valoarea PPI configurată pentru exportul imaginelor în setările Notemd. SVG rămâne o imagine vectorială și nu este influențată de valoarea PPI.

## Diagrame electrice

Pentru cereri de diagrame electrice suportate, alegeți **Diagramă electrică** cu **Fișier sursă CircuitikZ**. Notemd generează mai întâi o descriere structurată a circuitului, apoi scrie codul sursă CircuitikZ. Acest lucru este mai fiabil decât a solicita modelului să scrie un cod LaTeX fără restricții.

Suportul actual pentru CircuitikZ este intenționat limitat. Amplificatoare NMOS cu sursă comună și blocuri CMOS suportate funcționează cel mai bine. Verificați întotdeauna previsualizarea și codul sursă înainte de a utiliza un circuit generat în lucrări tehnice sau publicații.

Previsualizarea SVG este generată de Notemd ca instrument de verificare. Ea nu constituie dovadă că o instalație locală de LaTeX sau TikZJax a compilat cu succes fișierul `.tex`.

## Soluționare a problemelor

### Rezultatul este un tip de diagramă greșit

Alegeți explicit tipul de diagramă în loc să folosiți opțiunea Detectare automată. Pentru circuite, alegeți **Diagramă electrică** și utilizați comanda **Genera diagramă**, nu comanda vechiță **Summarise as Mermaid diagram**.

### Un format sursă nu este potrivit

Returnați formatul sursă la **Auto**, sau alegeți un format destinat acelui tip de diagramă. CircuitikZ este destinat diagramelor electrice; Draw.io și Drawnix sunt destinați diagramelor structurate suportate.

### Butoanele de export nu sunt vizibile

SVG, PNG și PDF necesită o previsualizare vizuală. Dacă există doar textul sursă, salvați fișierul sursă și deschideți-l în instrumentul extern corespunzător.

### Un fișier CircuitikZ nu se compilează în afara programului

Verificați dacă mediul LaTeX al dumneavoastră include pachetul `circuitikz`. Plugin-ul nu include o distribuție completă de LaTeX.
