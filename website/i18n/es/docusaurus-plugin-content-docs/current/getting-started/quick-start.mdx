# Guía de inicio rápido

<TLDR>
**Comience a usar Notemd en 5 minutos.** Tres pasos: (1) Agregue una clave LLM API (o utilice Ollama local), (2) Abra una nota y ejecute “Procesar archivo (agregar enlaces)”, (3) Explore flujos de trabajo con un clic en la barra lateral. Soporta más de 30 proveedores. No se requiere programación. Funciona sin conexión con Ollama.
</TLDR>

## Paso 1: Configure su primer proveedor LLM (2 min)

### Opción A: Proveedor de nube (OpenAI, Anthropic, etc.)

1. Abre **Ajustes** → **Notemd**
2. Bajo **LLM Configuración del proveedor**, seleccione un proveedor en el menú desplegable
3. Ingresa tu **API Key**
4. (Opcional) Haga clic en **"Obtener lista de modelos"** para ver los modelos disponibles
5. Seleccione un modelo para cada tipo de tarea

**Opciones populares:**
- **OpenAI**: `gpt-4o-mini` (rápido, económico)
- **Anthropic**: `claude-sonnet-4-6` (alta calidad)
- **DeepSeek**: `deepseek-chat` (muy económico)

### Opción B: Local Ollama (No se necesita clave API)

1. Instalar [Ollama](https://ollama.com/)
2. Descargar un modelo: `ollama pull llama3.1`
3. En la configuración de Notemd, seleccione **"Ollama"** como proveedor
4. Nombre del modelo: `llama3.1` (o cualquier modelo que hayas descargado)

## Paso 2: Pruebe su primera tarea (1 minuto)

### Añadir enlaces de Wiki a una nota

1. Abre cualquier nota con contenido significativo
2. Haga clic con el botón derecho en el editor
3. Seleccione **"Procesar archivo (agregar enlaces)"**
4. Espera unos segundos.
5. ¡Los conceptos clave ahora son `[[wiki-linked]]`!

**Ejemplo:**

**Antes:**
```
Machine learning models use neural networks to process data.
```

**Después:**
```
[[Machine learning]] models use [[neural networks]] to process data.
```

## Paso 3: Explorar funciones (2 min)

### Extraer notas de concepto

1. Hacer clic con el botón derecho → **"Extraer conceptos del archivo actual"**
2. Notemd crea archivos separados para cada concepto
3. Verifique la carpeta de conceptos configurada (por defecto: `concepts/`)
4. Cada nota conceptual tiene un enlace de retroceso a tu nota original

### Investigar y resumir

1. Resalta un tema en tu nota
2. Hacer clic con el botón derecho → **"Investigar y resumir"**
3. Notemd realiza búsquedas en la web (Tavily o DuckDuckGo)
4. El resumen se agrega al final de tu nota junto con las fuentes

### Flujos de trabajo de un clic

1. Revisa la **barra lateral izquierda** en busca de los botones del flujo de trabajo
2. Haga clic en **"Extraer con un clic"** en cualquier nota
3. Funciona de la siguiente manera: agregar enlaces → extraer conceptos → realizar investigación → generar diagrama
4. ¡Todo en un solo clic!

## Tareas comunes iniciales

### Tarea 1: Procesar un documento/artículo

```
1. Paste article content into a note
2. Run "Process file (add links)"
3. Run "Extract concepts"
4. Run "Research and summarize" on unfamiliar terms
5. Run "Summarize as Mermaid diagram" for visual structure
```

### Tarea 2: Traducir notas

```
1. Highlight text to translate
2. Right-click → "Translate to target language"
3. Set target language in settings first
```

### Tarea 3: Crear un mapa de conocimiento

```
1. Process 5-10 related notes with "add links"
2. Run "Extract concepts" on each
3. Open Graph view to see connections
4. Your knowledge graph is forming!
```

## Consejos de personalización

### Selección de modelo por tarea

Puedes usar modelos diferentes para tareas distintas:

| Tarea | Modelo recomendado | ¿Por qué? |
|------|------------------|-----|
| Agregar enlaces | `gpt-4o-mini`, `deepseek-chat` | Rápido, muchas llamadas |
| Extraer conceptos | `claude-sonnet-4-6`, `gpt-4o` | Extracción de calidad |
| Investigación | `perplexity`, `gpt-4o` | Basado en la web |
| Traducción | `gemini-flash`, `gpt-4o` | Multilingüe |
| Diagramas | `claude-opus`, modelo local | Razonamiento complejo |

Configurar en: **Ajustes → Notemd → Modelos específicos de tarea**

### Flujos de trabajo personalizados

Crea tu propio botón:

1. Ajustes → Notemd → Flujos de trabajo
2. Haz clic en "Agregar flujo de trabajo".
3. Ingresa DSL: `process-current-add-links>extract-concepts-current>research-and-summarize`
4. Nombre: "Mi flujo de investigación"
5. Aparece un nuevo botón en la barra lateral

## Solución de problemas

### "API Clave inválida"
- Verificar si la clave es correcta (sin espacios adicionales)
- Verificar si la cuenta tiene créditos
- Prueba con el botón "Prueba de conexión" en la configuración

### "Modelo no encontrado"
- Haga clic en "Obtener lista de modelos" para ver los modelos disponibles
- Algunos proveedores requieren nombres de modelo específicos (por ejemplo, `gpt-4o` y no `gpt-4`).

### Los enlaces no aparecen
- Asegúrese de que la nota tenga suficiente contenido (>50 palabras).
- Prueba un modelo más potente
- Verificar si la llamada a LLM tuvo éxito (ver panel de diagnóstico).

### Rendimiento lento
- Utilice modelos más rápidos (`gpt-4o-mini` en lugar de `gpt-4o`)
- Reduce el tamaño del bloque en la configuración
- Habilitar el procesamiento en paralelo para tareas por lotes
