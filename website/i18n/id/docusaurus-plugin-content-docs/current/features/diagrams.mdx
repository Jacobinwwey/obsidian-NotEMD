# Diagrams

<TLDR>
Pilih **jenis diagram yang Anda butuhkan**, lalu pilih **format sumber yang dapat diedit** untuknya. Lihat pratinjau hasilnya di Obsidian, kemudian ekspor sebagai SVG, PNG, atau PDF.
</TLDR>

## Pahami tiga pilihan yang tersedia

| Pilihan | Yang dikendalikan | Contoh |
|---|---|---|
| Jenis diagram | Makna dari diagram tersebut | Diagram alur, diagram sekuens, diagram sirkuit, grafik data |
| Format sumber | File yang dapat diedit atau digunakan kembali yang dibuat oleh Notemd | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| Format ekspor | Gambar atau dokumen yang digunakan untuk berbagi | SVG, PNG, PDF |

Draw.io dan Drawnix merupakan **format sumber**, bukan jenis diagram. CircuitikZ adalah format sumber yang dirancang khusus untuk diagram sirkuit. Pratinjau yang sama dapat diekspor ke beberapa format visual, sehingga memilih CircuitikZ tidak membatasi Anda hanya pada format SVG.

## Pilih format sumber

| Format | Paling cocok untuk | File sumber |
|---|---|---|
| Mermaid | Diagram yang disimpan langsung di dalam Markdown | Blok kode Mermaid |
| JSON Canvas | Proses kerja menggunakan canvas bawaan Obsidian | `.canvas` |
| Draw.io | Mengedit lebih lanjut di diagrams.net atau alat serupa | `.drawio` |
| Drawnix | Pengeditan bergaya gambar tangan atau papan tulis | `.drawnix` |
| CircuitikZ | Rangkaian listrik dan transistor yang terstruktur | `.tex` |

Biarkan opsi **Format sumber favorit** pada nilai **Auto** jika Anda tidak memerlukan editor tertentu. Notemd akan memilih format yang kompatibel untuk jenis diagram yang dipilih.

## Buat dan ekspor diagram

1. Buka catatan atau pilih teks yang ingin Anda visualisasikan.
2. Pilih opsi **Buat diagram** dari sidebar Notemd atau panel perintah.
3. Pilih **jenis diagram**, atau biarkan pada opsi **Deteksi otomatis**.
4. Pilih **format sumber**, atau biarkan pada opsi **Auto**.
5. Periksa pratinjau hasilnya.
6. Pilih opsi **Simpan file sumber**, **Ekspor SVG**, **Ekspor PNG**, atau **Ekspor PDF**.

PNG dan PDF menggunakan nilai PPI ekspor gambar yang telah dikonfigurasi di pengaturan Notemd. SVG tetap merupakan gambar vektor dan tidak terpengaruh oleh nilai PPI.

## Diagram sirkuit

Pilih opsi **Diagram sirkuit** dengan **file sumber CircuitikZ** untuk permintaan diagram sirkuit yang didukung. Notemd akan membuat deskripsi rangkaian yang terstruktur terlebih dahulu, lalu menulis kode sumber berformat CircuitikZ. Cara ini lebih dapat diandalkan dibandingkan meminta model untuk menulis kode LaTeX secara bebas.

Dukungan untuk CircuitikZ saat ini sengaja dibatasi. Penguat NMOS berbasis sumber umum dan blok CMOS yang didukung merupakan pilihan terbaik. Selalu periksa pratinjau dan kode sumber sebelum menggunakan rangkaian yang dihasilkan dalam pekerjaan teknis atau publikasi.

Pratinjau berformat SVG hanyalah alat bantu peninjauan yang dibuat oleh Notemd. Pratinjau tersebut bukan bukti bahwa instalasi LaTeX atau TikZJax lokal telah berhasil mengompilasi file berformat `.tex`.

## Penyelesaian masalah

### Hasilnya berupa jenis diagram yang salah

Pilih jenis diagram secara eksplisit, bukan menggunakan opsi Deteksi otomatis. Untuk diagram sirkuit, pilih opsi **Diagram sirkuit** dan gunakan perintah **Buat diagram**, bukan perintah lama **Ringkas sebagai diagram Mermaid**.

### Format sumber tidak cocok

Kembalikan format sumber ke nilai **Auto**, atau pilih format yang sesuai untuk jenis diagram tersebut. CircuitikZ hanya digunakan untuk diagram sirkuit; Draw.io dan Drawnix dirancang khusus untuk diagram terstruktur yang didukung.

### Tombol ekspor tidak muncul

SVG, PNG, dan PDF memerlukan pratinjau visual. Jika hanya teks sumber yang tersedia, simpan file sumber tersebut lalu buka di alat eksternal yang sesuai.

### File berformat CircuitikZ tidak dapat dikompilasi di luar aplikasi

Periksa apakah lingkungan LaTeX Anda sudah mencakup paket `circuitikz`. Plugin ini tidak menyertakan distribusi LaTeX yang lengkap.
