# 图表

<TLDR>
先选择**需要什么图表**，再选择它的**可编辑源格式**。在 Obsidian 中预览结果后，可导出为 SVG、PNG 或 PDF。
</TLDR>

## 理解三个不同选择

| 选择 | 决定什么 | 示例 |
|---|---|---|
| 图表类型 | 图表表达的含义 | 流程图、时序图、电路图、数据图表 |
| 生成格式 | Notemd 创建的可编辑或可复用源文件 | Mermaid、JSON Canvas、Draw.io、Drawnix、CircuitikZ |
| 导出格式 | 用于分享的图像或文档 | SVG、PNG、PDF |

Draw.io 和 Drawnix 是**源格式**，不是图表类型。CircuitikZ 是面向电路图的源格式。同一个预览可以导出为多种视觉格式，因此选择 CircuitikZ 并不意味着只能使用 SVG。

## 选择生成格式

| 格式 | 适用场景 | 源文件 |
|---|---|---|
| Mermaid | 把图表直接保存在 Markdown 中 | Mermaid 代码块 |
| JSON Canvas | Obsidian 原生画布工作流 | `.canvas` |
| Draw.io | 后续在 diagrams.net 或兼容工具中编辑 | `.drawio` |
| Drawnix | 手绘或白板风格编辑 | `.drawnix` |
| CircuitikZ | 结构化电路和晶体管电路 | `.tex` |

如果不需要指定外部编辑器，请把**首选生成格式**保留为**自动**。Notemd 会为所选图表类型选择兼容格式。

## 生成并导出图表

1. 打开要可视化的笔记，或选中相关文本。
2. 从 Notemd 侧栏或命令面板选择**生成图形**。
3. 选择**图表类型**，也可以保留为**自动检测**。
4. 选择**生成格式**，也可以保留为**自动**。
5. 检查预览结果。
6. 点击**导出**，再选择 SVG、PNG 或 PDF；如需保留可编辑产物，再使用**保存源文件**。

在预览中点击**历史**，可以在不遮蔽当前图形的情况下打开 Vault 历史抽屉。你可以搜索和筛选已完成图形、重新打开预览或打开已保存产物。即使当前没有活动笔记，也能从命令面板或 Notemd 侧边栏直接进入图形历史。

PNG 和 PDF 使用 Notemd 设置中的图像导出 PPI。SVG 保持矢量格式，不受 PPI 设置影响。

## 电路图

对于支持的电路请求，请选择**电路图**和 **CircuitikZ 源文件**。Notemd 会先生成结构化电路描述，再写出 CircuitikZ 源码。这比让模型直接生成不受约束的 LaTeX 更可靠。

当前 CircuitikZ 支持有意保持受约束范围。共源 NMOS 放大器和已支持的 CMOS 基础电路效果最佳。将生成电路用于技术资料或出版物前，请检查预览和源文件。

SVG 预览是 Notemd 生成的审阅辅助文件，不代表本机 LaTeX 或 TikZJax 已成功编译 `.tex` 文件。

**可选的原生编译。** 插件内预览以及 SVG、PNG、预览 PDF 导出不需要 LaTeX。桌面端如需编译诊断或原生 LaTeX PDF，可从图形设置或命令面板打开 **CircuitikZ 原生编译环境**。Notemd 可以复用系统 `tectonic` / `pdflatex`、使用自定义可执行文件，或在你明确点击**安装**后部署固定版本的托管 Tectonic。环境面板会在原位显示进度和错误，也支持稍后删除托管运行时。

## 常见问题

### 生成了错误的图表类型

请显式选择图表类型，不要使用自动检测。生成电路时应选择**电路图**并使用**生成图形**命令，不要使用旧的**总结为 Mermaid 图表**命令。

### 生成格式不适合当前图表

把生成格式恢复为**自动**，或者选择适合该图表类型的格式。CircuitikZ 用于电路图；Draw.io 和 Drawnix 用于其支持的结构化图表。

### 没有显示导出菜单

SVG、PNG 和 PDF 需要可用的视觉预览。如果当前只有源代码，请先保存源文件，再用对应的外部工具打开。

### CircuitikZ 文件在外部无法编译

请打开 **CircuitikZ 原生编译环境**并执行环境检查。可修复托管运行时，或确认自定义/系统编译器能够取得 `circuitikz` 包。Notemd 不会把完整 TeX 发行版捆绑进插件文件。
