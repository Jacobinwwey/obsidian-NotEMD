# 安装

<TLDR>
**2 分钟内安装 Notemd。** 推荐方式是 Obsidian Community Plugins，一键安装；也可以从 GitHub Releases 手动安装。需要 Obsidian 0.15.0+。支持 Windows、macOS、Linux、iOS 和 Android。插件无外部依赖，MIT 许可。
</TLDR>

## 方法 1：社区插件（推荐）

1. 打开 **Obsidian Settings**（齿轮图标，或 `Ctrl/Cmd + ,`）
2. 进入 **Community plugins**
3. 点击 **Browse**
4. 搜索 **“Notemd”**
5. 点击 **Install**
6. 点击 **Enable**

完成后，左侧边栏会出现 Notemd 图标。

## 方法 2：手动安装

### 准备条件

- Obsidian 0.15.0 或更高版本
- Git（可选，仅在克隆仓库时需要）

### 安装：工作流与任务

1. 从 [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest) 下载：
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. 找到 vault 的插件目录：

   ```text
   <your-vault>/.obsidian/plugins/
   ```

   如果 `plugins` 文件夹不存在，先创建它。

3. 创建 Notemd 插件目录：

   ```text
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. 把三个文件复制到 `notemd` 文件夹。

5. 重新加载 Obsidian：
   - 按 `Ctrl/Cmd + R`，或
   - 重启 Obsidian

6. 启用插件：
   - Settings → Community plugins
   - 在列表中找到 “Notemd”
   - 打开开关

## 替代方式：通过 BRAT 安装

如果你使用 [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat)：

1. 打开 BRAT 设置
2. 点击 “Add Beta plugin”
3. 输入：`Jacobinwwey/obsidian-NotEMD`
4. 点击 “Add Plugin”
5. 在 Community plugins 中启用 Notemd

## 安装：后续建议

安装后可以这样确认 Notemd 正常工作：

1. 打开任意笔记
2. 在编辑器中右键
3. 应能看到 Notemd 命令，例如：
   - “Process file (add links)”
   - “Extract concepts”
   - “Research and summarize”

## 卸载

移除 Notemd：

1. Settings → Community plugins
2. 找到 “Notemd”
3. 点击移除图标
4. 如需彻底清理，可删除插件目录：`<vault>/.obsidian/plugins/notemd/`

你的笔记、概念笔记等数据不会被删除。

## 安装：核心说明

### 自动更新（推荐）

Obsidian 会自动检查插件更新。有新版本时：

1. Settings → Community plugins
2. 点击 “Check for updates”
3. 点击 Notemd 旁边的 “Update”

### 手动更新

下载最新 release，替换插件目录中的三个文件。
