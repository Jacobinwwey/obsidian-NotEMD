# 安装

<TLDR>
**只需2分钟即可安装 Notemd。** 提供两种方式：(1) 使用 Obsidian 社区插件（推荐，一键安装），或 (2) 通过 GitHub Releases 进行手动安装。需要 Obsidian 0.15.0+ 版本。支持 Windows、macOS、Linux、iOS 和 Android 系统，无外部依赖。已有4000多次下载，采用 MIT 许可证。
</TLDR>

## 方法1：社区插件（推荐）

1. 打开**Obsidian 设置**（齿轮图标或 `Ctrl/Cmd + ,`）
2. 导航至**社区插件**
3. 点击**浏览**
4. 搜索 **“Notemd"**
5. 点击**安装**
6. 点击**启用**

**完成！** 您会在左侧侧边栏中看到 Notemd 图标。

## 方法2：手动安装

### 先决条件
- Obsidian 0.15.0 或更高版本
- Git（可选，用于克隆）

### 步骤

1. 从 [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest) **下载文件**：
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. **找到您的保险库插件文件夹：**
   ```
   <your-vault>/.obsidian/plugins/
   ```

如果 `plugins` 文件夹不存在，请创建它。

3. **创建 Notemd 文件夹：**
   ```
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. 将这三个文件复制到 `notemd` 文件夹中。

5. **重新加载 Obsidian：**
   - 按下 `Ctrl/Cmd + R`，或
   - 重启 Obsidian

6. **启用插件：**
   - 设置 → 社区插件
   - 在列表中查找“Notemd”
   - 打开它。

## 备选方案：通过 BRAT 安装

如果您使用 [BRAT 插件](https://github.com/TfTHacker/obsidian42-brat)：

1. 打开 BRAT 设置
2. 点击“添加测试版插件”
3. 输入：`Jacobinwwey/obsidian-NotEMD`
4. 点击“添加插件”
5. 在社区插件中启用 Notemd

## 验证

安装完成后，验证 Notemd 是否正常工作：

1. 打开任意笔记
2. 在编辑器中右键点击
3. 你应该会看到类似 Notemd 的命令：
   - “处理文件（添加链接）”
   - “提取概念”
   - “进行研究并总结”

## 卸载

要删除 Notemd：

1. 设置 → 社区插件
2. 查找“Notemd”
3. 点击 ❌ 图标
4. 如需，可删除插件文件夹：`<vault>/.obsidian/plugins/notemd/`

您的数据（笔记、概念说明等）将保持原样，不会被修改。

## 正在更新

### 自动更新（推荐）
Obsidian 会自动检查插件更新。当有新版本可用时：
1. 设置 → 社区插件
2. 点击“检查更新”
3. 点击 Notemd 旁边的“更新”按钮

### 手动更新
下载最新版本，然后替换插件文件夹中的三个文件。
