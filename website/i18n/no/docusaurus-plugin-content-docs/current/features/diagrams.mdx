# 图表

<TLDR>
首先确定**需要哪种类型的图表**，接着选择其**可编辑的源文件格式**。在 Obsidian 中预览效果后，即可将其导出为 SVG、PNG 或 PDF 格式。
</TLDR>

## 了解三种可选方案

| 选项 | 控制的内容 | 示例 |
|---|---|---|
| 图表类型 | 图表的含义 | 流程图、时序图、电路图、数据图表 |
| 源文件格式 | Notemd 生成的、可编辑或可重复使用的文件 | Mermaid、JSON Canvas、Draw.io、Drawnix、CircuitikZ |
| 导出格式 | 用于分享的图像或文档格式 | SVG、PNG、PDF |

Draw.io 和 Drawnix 属于**源文件格式**，而非图表类型。CircuitikZ 是专为电路图设计的源文件格式。同一预览内容可以导出为多种视觉格式，因此选择 CircuitikZ 并不会限制你只能得到 SVG 格式。

## 选择源文件格式

| 格式 | 最适合的场景 | 源文件 |
|---|---|---|
| Mermaid | 直接保存在 Markdown 中的图表 | Mermaid 代码块 |
| JSON Canvas | 基于 Obsidian 原生画布的工作流程 | `.canvas` |
| Draw.io | 后续在 diagrams.net 或兼容工具中进行编辑 | `.drawio` |
| Drawnix | 手绘或白板风格的编辑 | `.drawnix` |
| CircuitikZ | 结构化的电气及晶体管电路 | `.tex` |

如果您不需要特定的编辑器，可将**首选源文件格式**设置为**自动选择**。Notemd 会自动为选定的图表类型挑选兼容的格式。

## 生成并导出图表

1. 打开笔记或选中想要可视化的文本。
2. 从 Notemd 侧边栏或命令面板中选择**生成图表**。
3. 选定一种**图表类型**，或者保持为**自动检测**。
4. 选定一种**源文件格式**，或者保持为**自动选择**。
5. 查看预览效果。
6. 选择**保存源文件**、**导出 SVG**、**导出 PNG** 或 **导出 PDF**。

PNG 和 PDF 会使用在 Notemd 设置中配置的图像导出像素密度值。SVG 为矢量图像，不会受到像素密度值的影响。

## 电路图

对于支持的电路相关需求，请选择**电路图**并搭配**CircuitikZ 源文件**。Notemd 会先生成结构化的电路描述，然后再编写 CircuitikZ 源代码。这种方式比让模型直接编写不受限制的 LaTeX 代码更为可靠。

目前的 CircuitikZ 支持功能是有意限制的。基于通用源码的 NMOS 放大器以及部分受支持的 CMOS 构件效果最佳。在将生成的电路用于技术文档或出版物之前，务必仔细检查预览效果及源代码。

SVG 预览是由 Notemd 生成的辅助查看工具，并不能证明本地安装的 LaTeX 或 TikZJax 已成功编译了对应的 `.tex` 文件。

## 故障排除

### 生成的结果是错误的图表类型

请手动明确选择图表类型，而非依赖自动检测功能。对于电路图，应选择**电路图**并使用**生成图表**命令，而非旧版的**总结为 Mermaid 图表**命令。

### 源文件格式不合适

将源文件格式恢复为**自动选择**，或者挑选适合该图表类型的格式。CircuitikZ 仅适用于电路图；Draw.io 和 Drawnix 则适用于特定类型的结构化图表。

### 缺少导出按钮

SVG、PNG 和 PDF 格式需要可视预览功能。如果仅有源代码文本，建议先保存源文件，再在对应的外部工具中打开处理。

### CircuitikZ 文件在外部环境中无法编译

请确认您的 LaTeX 环境已包含 `circuitikz` 包。该插件并未内置完整的 LaTeX 发行版。
