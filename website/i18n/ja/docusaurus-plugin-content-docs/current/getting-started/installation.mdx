# インストール

<TLDR>
**2分以内にNotemdをインストールできます。** 2つの方法があります：(1) Obsidianコミュニティプラグイン（おすすめ、ワンクリックインストール）、または(2)GitHub Releases経由の手動インストールです。Obsidian 0.15.0+が必要です。Windows、macOS、Linux、iOS、Androidで動作します。外部依存関係はありません。4k回以上ダウンロードされており、MITライセンスが適用されています。
</TLDR>

## 方法1：コミュニティプラグイン（推奨）

1. **Obsidian 設定**を開く（⚙️アイコンまたは`Ctrl/Cmd + ,`）
2. **Community plugins**に移動します
3. **Browse**をクリックしてください
4. **"Notemd"**を検索する
5. **インストール**をクリックしてください
6. **Enable**をクリックしてください

**完了しました！** 左側のサイドバーにNotemdアイコンが表示されます。

## 方法2：手動インストール

### 前提条件
- Obsidian 0.15.0以上
- Git（クローン用、任意）

### 手順

1. [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest)から**ファイルをダウンロード**します：
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. **お使いのヴォルトのプラグインフォルダを探すには：**
   ```
   <your-vault>/.obsidian/plugins/
   ```

`plugins`フォルダが存在しない場合は、作成してください。

3. **Notemd フォルダを作成する:**
   ```
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. **3つのファイルを** `notemd` フォルダにコピーしてください。

5. **Obsidianを再読み込みします:**
   - `Ctrl/Cmd + R`を押すか、または
   - Obsidianを再起動してください

6. **プラグインを有効にする：**
   - 設定 → コミュニティプラグイン
   - リスト内の“Notemd”を探してください
   - オンに切り替えてください。

## 代替案：BRATを通じてインストールする

[BRAT プラグイン](https://github.com/TfTHacker/obsidian42-brat)を使用する場合：

1. BRATの設定を開く
2. 「Betaプラグインを追加」をクリックしてください。
3. 入力してください：`Jacobinwwey/obsidian-NotEMD`
4. 「プラグインを追加」をクリックしてください。
5. CommunityプラグインでNotemdを有効にする

## 検証

インストール後、Notemdが正常に動作しているか確認してください：

1. 任意のノートを開く
2. エディタで右クリックします
3. Notemdのコマンドは次のように表示されるはずです：
   - ファイルを処理する（リンクを追加）
   - 「概念を抽出する」
   - 「調査して要約する」

## アンインストール

Notemdを削除するには：

1. 設定 → コミュニティプラグイン
2. Notemdを探す
3. ❌アイコンをクリックしてください
4. 必要に応じて、プラグインフォルダを削除してください：`<vault>/.obsidian/plugins/notemd/`

お客様のデータ（メモ、コンセプトノートなど）はそのまま変更されません。

## 更新中

### 自動更新（推奨）
Obsidianは自動的にプラグインの更新をチェックします。新しいバージョンが利用可能な場合：
1. 設定 → コミュニティプラグイン
2. 「アップデートの確認」をクリックしてください。
3. Notemdの横にある「更新」をクリックしてください。

### 手動でのアップデート
最新版をダウンロードし、プラグインフォルダ内の3つのファイルを置き換えてください。
