# Diagrams

<TLDR>
உங்களுக்குத் தேவையான **டயகிராமின் வகையை** தேர்ந்தெடுத்து, அதற்கான **திருத்தக்கூடிய மூல வடிவத்தை** தேர்ந்தெடுக்கவும். Obsidian-இல் முன்னோட்டத்தைப் பார்த்து, அதை SVG, PNG அல்லது PDF வடிவத்தில் ஏற்றுமதி செய்யவும்.
</TLDR>

## மூன்று தேர்வுகளைப் புரிந்துகொள்ளுங்கள்

| தேர்வு | அது எதைக் கட்டுப்படுத்துகிறது | உதாரணங்கள் |
|---|---|---|
| டயகிராம் வகை | டயகிராம் என்ன அர்த்தம் கொண்டுள்ளது | Flowchart, sequence diagram, circuit diagram, data chart |
| மூல வடிவம் | Notemd உருவாக்கும் திருத்தக்கூடிய அல்லது மீண்டும் பயன்படுத்தக்கூடிய கோப்பு | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| ஏற்றுமதி வடிவம் | பகிர்வுக்காகப் பயன்படுத்தப்படும் படம் அல்லது ஆவணம் | SVG, PNG, PDF |

Draw.io மற்றும் Drawnix ஆகியவை **மூல வடிவங்கள்** தான்; டயகிராம் வகைகள் அல்ல. CircuitikZ என்பது சர்க்யூட் டயகிராம்களுக்காக வடிவமைக்கப்பட்ட ஒரு மூல வடிவம் ஆகும். ஒரே முன்னோட்டத்தைப் பல்வேறு காட்சி வடிவங்களில் ஏற்றுமதி செய்ய முடியும்; எனவே CircuitikZ-ஐத் தேர்ந்தெடுத்தாலும் SVG-க்கு மட்டுமே கட்டுப்படுவீர்கள் என்று அர்த்தமில்லை.

## ஒரு மூல வடிவத்தைத் தேர்ந்தெடுங்கள்

| வடிவம் | எதற்கு சிறந்தது | மூல கோப்பு |
|---|---|---|
| Mermaid | Markdown-இலேயே நேரடியாக வைக்கப்படும் டயகிராம்களுக்கு | Mermaid கோடுகள் |
| JSON Canvas | Obsidian-இன் உள்ளூர் canvas வேலைப்பாடுகளுக்கு | `.canvas` |
| Draw.io | diagrams.net அல்லது அதற்கு ஒத்த கருவிகளில் பின்னர் திருத்துவதற்கு | `.drawio` |
| Drawnix | கையால் வரையப்பட்ட அல்லது வைட்போர்ட்-போன்ற திருத்தங்களுக்கு | `.drawnix` |
| CircuitikZ | கட்டமைக்கப்பட்ட மின்சார மற்றும் டிரான்சிஸ்டர் சர்க்யூட்களுக்கு | `.tex` |

நீங்கள் குறிப்பிட்ட எந்த எடிட்டரையும் தேவைப்படுத்தவில்லை என்றால், **Preferred source format**-ஐ **Auto** என்ற நிலையிலேயே விட்டுவிடுங்கள். தேர்ந்தெடுக்கப்பட்ட டயகிராம் வகைக்கு ஏற்ற ஒரு பொருந்தக்கூடிய வடிவத்தை Notemd தானாகவே தேர்ந்தெடுக்கும்.

## ஒரு டயகிராமை உருவாக்கி ஏற்றுமதி செய்யுங்கள்

1. நோட்டைத் திறக்கவும் அல்லது நீங்கள் விசுவலைஸ் செய்ய விரும்பும் உரையைத் தேர்ந்தெடுக்கவும்.
2. Notemd சைட்பார் அல்லது கமாண்ட் பேலெட்டிலிருந்து **Generate diagram** என்பதைத் தேர்ந்தெடுக்கவும்.
3. ஒரு **டயகிராம் வகையை** தேர்ந்தெடுக்கவும், அல்லது அதை **Auto-detect** நிலையிலேயே விட்டுவிடவும்.
4. ஒரு **மூல வடிவத்தை** தேர்ந்தெடுக்கவும், அல்லது அதை **Auto** நிலையிலேயே விட்டுவிடவும்.
5. முன்னோட்டத்தை ஆய்வு செய்யவும்.
6. **Save source file**, **Export SVG**, **Export PNG** அல்லது **Export PDF** என்பதைத் தேர்ந்தெடுக்கவும்.

PNG மற்றும் PDF-கள், Notemd அமைப்புகளில் அமைக்கப்பட்டுள்ள படம் ஏற்றுமதி PPI-ஐப் பயன்படுத்துகின்றன. SVG ஒரு வெக்டர் படமாக இருப்பதால், PPI மதிப்பால் அது பாதிக்கப்படுவதில்லை.

## சர்க்யூட் டயகிராம்கள்

ஆதரிக்கப்படும் சர்க்யூட் தொடர்பான கோரிக்கைகளுக்கு, **Circuit diagram**-ஐ **CircuitikZ மூல கோப்பு** உடன் தேர்ந்தெடுக்கவும். Notemd முதலில் ஒரு கட்டமைக்கப்பட்ட சர்க்யூட் விவரணையை உருவாக்கி, பின்னர் CircuitikZ மூலத்தை எழுதுகிறது. இது, மாடலை கட்டுப்பாடுகள் இல்லாமல் LaTeX-ஐ எழுதச் சொல்வதை விட மிகவும் நம்பகமானது.

தற்போதைய CircuitikZ ஆதரவு தேவையான அளவுக்கு மட்டுமே வரையறுக்கப்பட்டுள்ளது. Common-source NMOS amplifiers மற்றும் ஆதரிக்கப்படும் CMOS building blocks தான் சிறந்த முடிவுகளைத் தரும். தொழில்நுட்ப அல்லது வெளியீட்டுப் பணிகளில் உருவாக்கப்பட்ட சர்க்யூட்டைப் பயன்படுத்துவதற்கு முன், எப்போதும் முன்னோட்டம் மற்றும் மூல கோப்பை ஆய்வு செய்யுங்கள்.

SVG முன்னோட்டம் என்பது Notemd உருவாக்கும் ஒரு ஆய்வு உதவியாகும். இது, உங்கள் உள்ளூர் LaTeX அல்லது TikZJax நிறுவல் `.tex` கோப்பை வெற்றிகரமாக கம்பைல் செய்துள்ளது என்பதற்கான சான்று அல்ல.

## பிரச்சினை நிவாரணம்

### முடிவு தவறான டயகிராம் வகையாக உள்ளது

Auto-detect-ஐப் பயன்படுத்துவதற்குப் பதிலாக, டயகிராம் வகையை நேரடியாகத் தேர்ந்தெடுக்கவும். சர்க்யூட் டயகிராமுக்கு, **Circuit diagram**-ஐத் தேர்ந்தெடுத்து, **Generate diagram** கட்டளையைப் பயன்படுத்தவும்; பழைய **Summarise as Mermaid diagram** கட்டளையைப் பயன்படுத்த வேண்டாம்.

### ஒரு மூல வடிவம் பொருந்தாது

மூல வடிவத்தை **Auto** நிலைக்கு மாற்றவும், அல்லது அந்த டயகிராம் வகைக்கு ஏற்ற வடிவத்தைத் தேர்ந்தெடுக்கவும். CircuitikZ என்பது சர்க்யூட் டயகிராம்களுக்காக மட்டுமே; Draw.io மற்றும் Drawnix ஆகியவை ஆதரிக்கப்படும் கட்டமைக்கப்பட்ட டயகிராம்களுக்காகவே உள்ளன.

### ஏற்றுமதி பொத்தான்கள் காணப்படவில்லை

SVG, PNG மற்றும் PDF-களுக்கு ஒரு காட்சி முன்னோட்டம் தேவை. மூல உரை மட்டுமே கிடைத்தால், மூல கோப்பைச் சேமித்து, அதை பொருந்தக்கூடிய வெளிப்புற கருவியில் திறக்கவும்.

### CircuitikZ கோப்பு வெளிப்புறத்தில் கம்பைல் ஆகவில்லை

உங்கள் LaTeX சூழலில் `circuitikz` package இருக்கிறதா என்பதைச் சரிபார்க்கவும். இந்த பிளக்இன் முழுமையான LaTeX விநியோகத்தை உள்ளடக்கியது அல்ல.
