# Diagrammes

<TLDR>
Choisissez **le type de diagramme dont vous avez besoin**, puis sélectionnez son **format source modifiable**. Consultez l’avant-visualisation dans Obsidian et exportez le fichier en SVG, PNG ou PDF.
</TLDR>

## Comprendre les trois options possibles

| Option | Ce qu’elle contrôle | Exemples |
|---|---|---|
| Type de diagramme | La signification du diagramme | Diagramme de flux, diagramme de séquence, schéma électrique, graphique de données |
| Format source | Le fichier modifiable ou réutilisable généré par Notemd | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| Format d’exportation | L’image ou le document utilisé pour le partage | SVG, PNG, PDF |

Draw.io et Drawnix sont des **formats sources**, pas des types de diagrammes. CircuitikZ est un format source conçu spécifiquement pour les schémas électriques. La même avant-visualisation peut être exportée sous plusieurs formats visuels ; par conséquent, choisir CircuitikZ ne vous limite pas à l’export en SVG.

## Choisir un format source

| Format | Meilleur usage | Fichier source |
|---|---|---|
| Mermaid | Diagrammes conservés directement dans le Markdown | Bloc de code Mermaid |
| JSON Canvas | Flux de travail natifs d’Obsidian basés sur le canvas | `.canvas` |
| Draw.io | Édition ultérieure dans diagrams.net ou des outils compatibles | `.drawio` |
| Drawnix | Édition manuscrite ou au style tableau blanc | `.drawnix` |
| CircuitikZ | Circuits électriques structurés et transistors | `.tex` |

Laissez l’option **Format source préféré** sur **Auto** si vous n’avez pas besoin d’un éditeur particulier. Notemd choisira alors un format compatible avec le type de diagramme sélectionné.

## Générer et exporter un diagramme

1. Ouvrez la note ou sélectionnez le texte que vous souhaitez visualiser.
2. Sélectionnez **Générer diagramme** dans la barre latérale de Notemd ou dans la palette de commandes.
3. Choisissez un **type de diagramme**, ou laissez l’option sur **Détection automatique**.
4. Sélectionnez un **format source**, ou laissez l’option sur **Auto**.
5. Consultez l’avant-visualisation.
6. Choisissez **Enregistrer le fichier source**, **Exporter en SVG**, **Exporter en PNG** ou **Exporter en PDF**.

PNG et PDF utilisent le taux de points par pouce (PPI) défini pour l’export d’image dans les paramètres de Notemd. L’SVG reste une image vectorielle et n’est donc pas affecté par cette valeur de PPI.

## Schémas électriques

Pour les demandes de schémas électriques pris en charge, choisissez **Schéma électrique** avec un **fichier source CircuitikZ**. Notemd génère d’abord une description structurée du circuit, puis écrit le code source CircuitikZ. Cette méthode est plus fiable que de demander au modèle d’écrire du LaTeX sans contraintes.

La compatibilité actuelle avec CircuitikZ est délibérément limitée. Les amplificateurs NMOS à source commune et les blocs CMOS pris en charge fonctionnent le mieux. Vérifiez toujours l’avant-visualisation ainsi que le code source avant d’utiliser un circuit généré dans des travaux techniques ou de publication.

L’avant-visualisation en SVG est une aide à la révision générée par Notemd. Elle ne constitue pas une preuve que le fichier `.tex` a été compilé avec succès via une installation locale de LaTeX ou TikZJax.

## Résolution de problèmes

### Le type de diagramme obtenu est incorrect

Choisissez explicitement le type de diagramme au lieu d’utiliser la détection automatique. Pour les schémas électriques, sélectionnez **Schéma électrique** et utilisez la commande **Générer diagramme**, plutôt que l’ancienne commande **Résumer en diagramme Mermaid**.

### Un format source n’est pas adapté

Rendez le format source à la valeur **Auto**, ou choisissez un format conçu pour ce type de diagramme. CircuitikZ est destiné aux schémas électriques ; Draw.io et Drawnix sont prévus pour les diagrammes structurés pris en charge.

### Les boutons d’exportation manquent

SVG, PNG et PDF nécessitent une avant-visualisation visuelle. Si seul du texte source est disponible, enregistrez le fichier source puis ouvrez-le dans l’outil externe correspondant.

### Un fichier CircuitikZ ne se compile pas à l’extérieur

Vérifiez que votre environnement LaTeX inclut le paquet `circuitikz`. Le plugin ne fournit pas une distribution complète de LaTeX.
