# Guide de démarrage rapide

<TLDR>
**Commencez à utiliser Notemd en 5 minutes.** Trois étapes : (1) Ajoutez une clé LLM API (ou utilisez Ollama local), (2) Ouvrez une note et exécutez « Process file (add links) », (3) Découvrez les workflows en un clic dans la barre latérale. Prend en charge plus de 30 fournisseurs. Aucun codage requis. Fonctionne hors ligne avec Ollama.
</TLDR>

## Étape 1 : Configurer votre premier fournisseur LLM (2 min)

### Option A : Fournisseur de cloud (OpenAI, Anthropic, etc.)

1. Ouvrez **Paramètres** → **Notemd**
2. Sous **LLM Configuration du fournisseur**, sélectionnez un fournisseur dans la liste déroulante
3. Entrez votre clé **API**
4. (Optionnel) Cliquez sur **« Obtenir la liste des modèles »** pour voir les modèles disponibles
5. Sélectionnez un modèle pour chaque type de tâche

**Choix populaires :**
- **OpenAI**: `gpt-4o-mini` (rapide, abordable)
- **Anthropic**: `claude-sonnet-4-6` (haute qualité)
- **DeepSeek**: `deepseek-chat` (très abordable)

### Option B : Local Ollama (Aucune clé API requise)

1. Installez [Ollama](https://ollama.com/)
2. Télécharger un modèle : `ollama pull llama3.1`
3. Dans les paramètres de Notemd, sélectionnez **"Ollama"** comme fournisseur
4. Nom du modèle : `llama3.1` (ou tout modèle que vous avez téléchargé)

## Étape 2 : Essayez votre première tâche (1 min)

### Ajouter des liens Wiki à une note

1. Ouvrez n’importe quelle note contenant du contenu significatif
2. Cliquez avec le bouton droit dans l’éditeur
3. Sélectionnez **« Traiter le fichier (ajouter des liens) »**
4. Attendez quelques secondes.
5. Les concepts clés sont maintenant `[[wiki-linked]]` !

**Exemple :**

**Avant :**
```
Machine learning models use neural networks to process data.
```

**Après :**
```
[[Machine learning]] models use [[neural networks]] to process data.
```

## Étape 3 : Découvrir les fonctionnalités (2 min)

### Extraire les notes de concept

1. Clic droit → **"Extraire les concepts du fichier actuel"**
2. Notemd crée des fichiers séparés pour chaque concept
3. Vérifiez votre dossier de concept configuré (par défaut : `concepts/`)
4. Chaque note de concept contient un lien vers votre note originale

### Recherche et résumé

1. Sélectionnez un sujet dans votre note
2. Clic droit → **« Rechercher et résumer »**
3. Notemd effectue des recherches sur le web (Tavily ou DuckDuckGo)
4. Résumé : ajoutez des sources à votre note

### Flux de travail en un clic

1. Vérifiez la **barre latérale gauche** pour les boutons de flux de travail
2. Cliquez sur **"Extraction en un clic"** dans n’importe quelle note
3. Il s’exécute ainsi : ajouter des liens → extraire des concepts → effectuer des recherches → générer un diagramme
4. Tout en un clic !

## Tâches courantes initiales

### Tâche 1 : Traiter un papier/un article

```
1. Paste article content into a note
2. Run "Process file (add links)"
3. Run "Extract concepts"
4. Run "Research and summarize" on unfamiliar terms
5. Run "Summarize as Mermaid diagram" for visual structure
```

### Tâche 2 : Traduire les notes

```
1. Highlight text to translate
2. Right-click → "Translate to target language"
3. Set target language in settings first
```

### Tâche 3 : Créer une carte de connaissances

```
1. Process 5-10 related notes with "add links"
2. Run "Extract concepts" on each
3. Open Graph view to see connections
4. Your knowledge graph is forming!
```

## Conseils de personnalisation

### Sélection du modèle par tâche

Vous pouvez utiliser des modèles différents pour des tâches différentes :

| Tâche | Modèle recommandé | Pourquoi |
|------|------------------|-----|
| Ajouter des liens | `gpt-4o-mini`, `deepseek-chat` | Vite, de nombreuses appels |
| Extraire les concepts | `claude-sonnet-4-6`, `gpt-4o` | Extraction de qualité |
| Recherche | `perplexity`, `gpt-4o` | Basé sur le Web |
| Traduction | `gemini-flash`, `gpt-4o` | Multilingue |
| Diagrammes | `claude-opus`, modèle local | Raisonnement complexe |

Configurer dans : **Paramètres → Notemd → Modèles spécifiques aux tâches**

### Flux de travail personnalisés

Créez votre propre bouton :

1. Paramètres → Notemd → Flux de travail
2. Cliquez sur « Ajouter un flux de travail »
3. Entrez le DSL : `process-current-add-links>extract-concepts-current>research-and-summarize`
4. Nommez-le : « Mon flux de recherche »
5. Un nouveau bouton apparaît dans la barre latérale

## Résolution de problèmes

### "Clé API invalide"
- Vérifier si la clé est correcte (pas d’espaces supplémentaires)
- Vérifier si le compte dispose de crédits
- Testez avec le bouton « Test de connexion » dans les paramètres

### "Modèle non trouvé"
- Cliquez sur « Obtenir la liste des modèles » pour voir les modèles disponibles
- Certains fournisseurs exigent des noms de modèle spécifiques (par exemple, `gpt-4o` et non `gpt-4`).

### Les liens ne s’affichent pas
- Assurez-vous que la note contient suffisamment de contenu (plus de 50 mots).
- Essayez un modèle plus puissant
- Vérifier si l’appel LLM a réussi (voir le panneau de diagnostic)

### Performance lente
- Utilisez des modèles plus rapides (`gpt-4o-mini` au lieu de `gpt-4o`)
- Réduire la taille du chunk dans les paramètres
- Activer le traitement parallèle pour les tâches en lot
