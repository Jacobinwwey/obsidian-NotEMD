# Installation

<TLDR>
**Installez Notemd en moins de 2 minutes.** Deux méthodes : (1) les plugins communautaires Obsidian (recommandés, installation en un clic), ou (2) installation manuelle via GitHub Releases. Nécessite Obsidian 0.15.0+. Fonctionne sous Windows, macOS, Linux, iOS et Android. Aucune dépendance externe. Plus de 4 000 téléchargements, sous licence MIT.
</TLDR>

## Méthode 1 : Plugins communautaires (recommandé)

1. Ouvrez les **Obsidian Paramètres** (icône ⚙️ ou `Ctrl/Cmd + ,`)
2. Aller vers **Plugins communautaires**
3. Cliquez sur **Parcourir**
4. Rechercher **"Notemd"**
5. Cliquez sur **Installer**
6. Cliquez sur **Activer**

**Fait !** Vous verrez l’icône Notemd dans votre barre latérale gauche.

## Méthode 2 : Installation manuelle

### Prérequis
- Obsidian 0.15.0 ou une version ultérieure
- Git (facultatif, pour le clonage)

### Étapes

1. **Télécharger des fichiers** depuis [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest) :
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. **Localisez le dossier des plugins de votre coffre-fort :**
   ```
   <your-vault>/.obsidian/plugins/
   ```

Si le dossier `plugins` n’existe pas, créez-le.

3. **Créer le dossier Notemd :**
   ```
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. **Copiez les trois fichiers** dans le dossier `notemd`.

5. **Recharger Obsidian:**
   - Appuyez sur `Ctrl/Cmd + R`, ou
   - Redémarrer Obsidian

6. **Activer le plugin :**
   - Paramètres → Plugins communautaires
   - Trouver "Notemd" dans la liste
   - Le mettre en marche

## Alternative : Installer via BRAT

Si vous utilisez le [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat) :

1. Ouvrir les paramètres de BRAT
2. Cliquez sur « Ajouter le plugin bêta ».
3. Entrer : `Jacobinwwey/obsidian-NotEMD`
4. Cliquez sur « Ajouter un plugin »
5. Activer Notemd dans les plugins communautaires

## Vérification

Après l'installation, vérifiez que Notemd fonctionne :

1. Ouvrir n’importe quelle note
2. Cliquez avec le bouton droit dans l’éditeur
3. Vous devriez voir des commandes Notemd telles que :
   - "Traiter le fichier (ajouter des liens)"
   - « Extraire les concepts »
   - « Rechercher et résumer »

## Désinstallation

Pour supprimer Notemd :

1. Paramètres → Plugins communautaires
2. Trouver Notemd
3. Cliquez sur l’icône ❌
4. Optionnellement, supprimez le dossier du plugin : `<vault>/.obsidian/plugins/notemd/`

Vos données (notes, notes de concept, etc.) restent inchangées.

## Mise à jour

### Mise à jour automatique (recommandé)
Obsidian vérifie automatiquement la présence de mises à jour de plugins. Lorsqu’une nouvelle version est disponible :
1. Paramètres → Plugins communautaires
2. Cliquez sur « Vérifier les mises à jour ».
3. Cliquez sur « Mettre à jour » à côté de Notemd

### Mise à jour manuelle
Téléchargez la dernière version et remplacez les trois fichiers dans votre dossier de plugin.
