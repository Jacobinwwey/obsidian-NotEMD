## Installation et configuration

### Comment installer Notemd dans Obsidian ?

Ouvrez Obsidian **Paramètres → Plugins communautaires → Parcourir** → recherchez **"Notemd"** → **Installer** → **Activer**.

**Exigences :**
- Obsidian 0.15.0+
- Compatible sur ordinateur de bureau (Windows, macOS, Linux) et mobile (iOS, Android)

**Installation manuelle :**
1. Téléchargez `main.js`, `styles.css`, `manifest.json` depuis [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases)
2. Placez-les dans `<your-vault>/.obsidian/plugins/notemd/`
3. Recharger Obsidian et activer le plugin

**Statistiques :** plus de 4 000 téléchargements, 44 versions publiées, sous licence MIT.

### Quels fournisseurs LLM Notemd prend-il en charge ?

Notemd prend en charge **plus de 30 fournisseurs** :

**Fournisseurs de cloud :**

- Anthropic (Claude Opus, Sonnet, Haiku)
- Google (Gemini Pro, Flash, Ultra)
- xAI (Grok)

**Axé sur la Chine :**

- Huawei Cloud MaaS, Baidu Qianfan, SiliconFlow

**Local/autohébergé :**
- Ollama

- Tout point de terminaison compatible avec OpenAI

**Passerelles :**
- Azure OpenAI, OpenRouter, Together IA, Groq, AIHubMix
- GitHub Models, Fireworks, LiteLLM, Nebius, Cerebras
- Hugging Face, Vercel AI Gateway, Requesty

### Notemd peut-il fonctionner avec des LLM locaux ?

**Oui, prise en charge complète.** Notemd fonctionne avec :
- **Ollama** (intégration native)
- **LM Studio** (via l’endpoint compatible OpenAI)
- **vLLM, LocalAI, text-generation-webui** (via le mode compatible OpenAI)

Toutes les tâches fonctionnent avec des modèles locaux : le liage, la génération de concepts, la traduction, la recherche et la génération de diagrammes.

**Exemple de mélange et d’association :**
- Ollama (local) pour le lienage → rapide, privé, sans coût
- OpenAI pour la recherche → haute qualité, intégration de recherche sur le Web
- Gemini pour la traduction → prise en charge multimodale

### Comment configurer des modèles différents pour des tâches différentes ?

Dans les **Notemd Paramètres**, chaque tâche dispose de sélectionneurs de fournisseur/modèle indépendants :

| Tâche | Configuration d’exemple |
|------|----------------------|
| **Lien** | DeepSeek (rapide, économique) |
| **Génération de concepts** | Claude Opus (haute qualité) |
| **Recherche** | Perplexity ou GPT-4o (basé sur le web) |
| **Traduction** | Gemini Flash (multimodal) |
| **Génération de diagrammes** | Ollama (local, privé) |

Cette optimisation par tâche équilibre la vitesse, le coût et la qualité.

## Fonctionnalités principales

### En quoi Notemd diffère-t-il de ChatGPT pour la lecture de documents papier ?

| Dimensions | ChatGPT | Notemd |
|-----------|---------|--------|
| **Emplacement de sortie** | Historique de chat (éphémère) | Obsidian coffre-fort (fichiers persistants) |
| **Format du résultat** | Réponse de texte | Liens wiki, notes de concept, diagrammes, traductions |
| **Valeur à long terme** | Disparaît après la session | S’accumule dans la base de connaissances |
| **Réutilisabilité** | Il faut relancer la requête | Les fichiers restent, le lien évolue |

**Utilisez ChatGPT pour :** Des questions-réponses rapides, des explications ponctuelles
**Utilisez Notemd pour :** Créer une base de connaissances persistante et structurée sur plusieurs semaines/mois

### Comment Notemd ajoute-t-il des liens wiki aux notes ?

**Processus :**
1. Vous exécutez « Process file (add links) » sur une note
2. Notemd envoie le texte vers votre LLM configuré
3. Le LLM identifie les concepts clés dans le contexte
4. Notemd insère `[[wiki-links]]` à chaque occurrence du concept
5. Optionnellement, il crée automatiquement des notes de concept dans un dossier que vous spécifiez

**Fonctionnalités :**
- Suppression des synonymes (évite les doublons entre « transformer » et « transformers »)
- Liens inversés depuis les notes de concept vers la source
- Intégrité du lien lors du renomage/effacement
- Mode d’extraction pure (extraire les concepts sans modifier l’original)

### Qu’est-ce que le flux de travail en un clic Notemd ?

Les **flux de travail en un clic** relient plusieurs actions en un **seul bouton dans la barre latérale**.

**Flux de travail par défaut :** `One-Click Extract`
```
process-current-add-links > extract-concepts-current > research-and-summarize > summarize-as-mermaid
```

**Résultat :** Un clic déclenche :
1. Ajouter des liens wiki à la note actuelle
2. Générer des notes de concept pour de nouveaux termes
3. Recherche et ajout d’informations de contexte
4. Comprimer en un diagramme Mermaid

**Flux de travail personnalisés :** Définissez les vôtres dans Paramètres en utilisant le format DSL. Chaque flux de travail devient un bouton de barre latérale réutilisable.

### En quoi Notemd se compare-t-il à Copilot ou Smart Connections ?

Ces plugins résolvent différents problèmes :

- **Copilot** — Priorité à la conversation. Vous discutez avec une IA de vos notes. Les réponses de l’IA restent dans le panneau de chat ; elles ne modifient pas votre coffre-fort.
- **Smart Connections** — Priorité à la découverte. Il trouve des notes sémantiquement similaires à l’aide d’embeddings. Il ne crée ni ne modifie de contenu.
- **Notemd** — Écrire en premier. L’IA traite vos notes et génère des résultats structurés (liens wiki, notes de concepts, recherches, diagrammes, traductions) qui sont directement enregistrés dans votre coffre-fort sous forme de fichiers persistants.

| Ce dont vous avez besoin | Plugin recommandé |
|----------------|-------------------|
| Discutez avec l’IA des notes | Copilot |
| Trouver des notes similaires ou liées | Smart Connections |
| Insérer automatiquement des liens wiki et créer un graphe de connaissances | **Notemd** |
| Générer des notes de concept avec des liens renvoyants | **Notemd** |
| Créer des diagrammes à partir du contenu des notes | **Notemd** |
| Recherchez un sujet et ajoutez des résumés | **Notemd** |

Les trois peuvent être utilisés ensemble : ils se complètent plutôt qu’ils ne se concurrencent.

## Recherche et traduction

### Quels services de recherche fonctionnent avec la recherche Notemd ?

| Service | API Une clé est requise | Qualité | Notes |
|---------|-----------------|---------|-------|
| **Tavily** | Oui | Élevé | Recommandé : recherche optimisée par l’IA |
| **DuckDuckGo** | Non | Expérimental | Pas de configuration nécessaire, qualité inférieure |

**Comment fonctionne « Recherche et résumé » :**
1. Vous sélectionnez un sujet ou vous le lancez sur la note actuelle
2. Notemd interroge le service de recherche
3. Résultats → résumé de LLM
4. Résumé ajouté à votre note avec des citations de source

**Résultat :** Un historique de recherche persistant dans votre coffre-fort, et non dans les onglets du navigateur.

### Notemd prend-il en charge le chinois ou d’autres langues ?

**Langues UI :** plus de 21 langues prises en charge
- Anglais, Chinois (simplifié), Chinois (traditionnel), Japonais, Coréen
- Arabe, Allemand, Français, Espagnol, Russe, Portugais, Italien
- Hindi, bengali, thaï, vietnamien, turc, polonais, tchèque et bien d’autres

**Langue de sortie de la tâche :** Configuration indépendante
- UI en chinois + résultat en anglais : ✅
- UI en anglais + sortie en chinois : ✅

**Cas d’usage :** Les chercheurs bilingues peuvent conserver UI dans leur langue maternelle tout en générant des notes en anglais pour publication.

## PDF et flux de travail académiques

### Puis-je utiliser Notemd pour la lecture sur papier/PDF ?

**Oui.** Flux de travail :
1. **Convertir PDF en Markdown** à l’aide de [MinerU](https://github.com/opendatalab/MinerU) (recommandé pour les articles académiques)
2. Ouvrez le fichier Markdown dans Obsidian
3. Exécuter les tâches Notemd :
   - **Ajouter des liens wiki** aux termes clés
   - **Générer des notes de concept** pour une terminologie inconnue
   - **Extraire le texte d’origine** à l’aide de questions personnalisées (par exemple : « Quelles sont les limites ? »)
   - **Contexte de recherche** pour les méthodes/ensembles de données cités
   - **Traduire** les sections difficiles
   - **Compacter en diagramme Mermaid** pour une compréhension structurelle

**Version 1.9.1+ :** Extraction de la structure du chapitre avec prise en charge de l’index.

**Résultat :** Votre lecture de document devient un artefact de connaissance structuré et lienable dans votre coffre-fort.

## Diagrammes et visualisation

### Quels types de graphes/diagrammes Notemd peut-il générer ?

| Taper | Assistance | Fonctionnalités |
|------|---------|----------|
| **Mermaid** | Complet | Diagramme de flux, séquence, classe, état, modèle entité-relation, diagramme de Gantt, correction automatique de syntaxe, validation par lots |
| **JSON Canvas** | Complet | Mise en page native du canvas Obsidian |
| **Vega-Lite** | Complet | Graphiques de données, séries temporelles, barres, points, personnalisables |
| **HTML** | Solution de secours | Pour les types de diagrammes non pris en charge |
| **HTML/SVG éditable** | Cible explicite | Inline autonome SVG avec des annotations de style Draw.io sémantiques |
| **Draw.io / Drawnix artefacts** | Exporter la frontière | Déterministe XML/JSON à partir de `SemanticFigureModel` ; aucune dépendance en temps de exécution vis-à-vis d’un éditeur |

**Mermaid auto-fix :** Mode de débogage avancé pour les réparations :
- Flèches brisées (`-->`, `-.->`, `==>`)
- Étiquettes et commentaires invalides
- Erreurs de syntaxe du sous-graphe
- Notation de forme (`[`, `(`, `{`, `((`)
- Caractères de pipe dans les étiquettes
- Notes des pièces jointes

**Flux de travail :**
1. Exécutez « Summarize as Mermaid diagram » sur une note
2. Aperçu dans une boîte modale
3. Exporter en SVG ou PNG
4. Enregistrer sous forme de fichier `.mmd` dans le coffre-fort

## Résolution de problèmes

### Mes appels LLM échouent. Que dois-je vérifier ?

1. **API la clé est-elle valide ?** Vérifiez Paramètres → Configuration du fournisseur
2. **Le nom du modèle est-il correct ?** Utilisez le bouton « 获取模型列表 » (Obtenir la liste des modèles) dans les paramètres
3. **Le réseau est-il accessible ?** Certains fournisseurs exigent un VPN
4. **Limites de vitesse ?** Vérifiez le tableau de bord du fournisseur
5. **Diagnostic :** Exécutez le « Test de connexion » dans les paramètres pour obtenir des journaux d’erreurs détaillés

### Notemd est lent. Comment puis-je l’accélérer ?

1. **Utiliser des modèles plus rapides :** DeepSeek, Gemini Flash, GPT-4o-mini
2. **Réduire la taille des blocs :** Paramètres → Fragmentation intelligente → diminuer le nombre de mots
3. **Traitement parallèle :** Paramètres → Traitement par lots → augmenter la concurrence
4. **Modèles locaux :** Ollama pour les tâches qui n'ont pas besoin de recherche sur le Web

### Notemd peut-il fonctionner à 100 % hors ligne ?

**Oui, avec Ollama.** Aucune clé API, pas d’internet, aucune donnée ne quitte votre machine :

1. Installez [Ollama](https://ollama.com) et exécutez `ollama serve`
2. Dans les paramètres de Notemd, sélectionnez **Ollama** comme votre fournisseur
3. Tirer un modèle : `ollama pull llama3` (ou qwen2.5, gemma2, etc.)
4. Toutes les fonctionnalités fonctionnent : liens wiki, notes de concept, traduction, diagrammes

**Recherche DuckDuckGo** fonctionne également sans clé API (qualité expérimentale).

Avec les fournisseurs de services cloud (OpenAI, Anthropic, Google, DeepSeek, etc.), le contenu de votre note est envoyé sous forme d’entrée API. Vérifiez la politique de confidentialité de chaque fournisseur pour en connaître les modalités de traitement des données.

### Mes données sont-elles envoyées dans le cloud ?

**Cela dépend de votre fournisseur :**

| Type de fournisseur | Les données sont-elles envoyées dans le cloud ? | Notes |
|--------------|---------------------|-------|
| **Ollama (local)** | Non | Tout reste sur votre ordinateur. |
| **LMStudio (local)** | Non | Identique à Ollama |
| **DeepSeek, OpenAI, etc.** | Oui | Contenu de la note envoyé en tant qu’entrée API pour traitement |
| **OpenRouter, AIHubMix** | Oui | Passerelle qui redirige vers les fournisseurs de cloud |

Notemd ne collecte **aucune donnée de télémétrie, aucune donnée analytique, aucun données d’utilisateur**. Le plugin est entièrement open source (MIT). Les seules données qui quittent votre ordinateur sont celles que vous envoyez explicitement à votre fournisseur LLM configuré.

### Quel plugin d’IA Obsidian est le meilleur pour la recherche ?

Cela dépend de votre flux de travail de recherche :

| Besoin | Plugin | Pourquoi |
|------|--------|-----|
| Trouver des articles/notes connexes | Smart Connections | Découverte de la similarité sémantique |
| Parlez de vos sources | Copilot | Conversation dans le contexte du coffre-fort |
| Lier automatiquement les concepts + générer des notes | **Notemd** | L’IA identifie les concepts, insère des liens et crée des notes |
| Recherche sur le Web + résumé en notes | **Notemd** | Tavily/DuckDuckGo Recherche → LLM Résumé → ajouté à la note |
| Générer des diagrammes à partir de recherches | **Notemd** | Mermaid/Canvas/Vega-Lite depuis le contenu de la note |

**Combinaison recommandée :** Smart Connections (découverte) + Notemd (structuration) + Copilot (discussion).
