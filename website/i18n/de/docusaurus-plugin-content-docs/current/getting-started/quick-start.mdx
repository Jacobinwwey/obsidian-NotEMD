# Leitfaden für eine schnelle Einrichtung

<TLDR>
**Beginnen Sie bereits in 5 Minuten mit der Verwendung von Notemd.** Drei Schritte: (1) Fügen Sie eine LLM API-Schlüssel hinzu (oder verwenden Sie einen lokalen Ollama), (2) Öffnen Sie eine Notiz und führen Sie „Datei verarbeiten (Verlinkungen hinzufügen)“ aus, (3) Entdecken Sie One-Click-Arbeitsabläufe im Seitenbereich. Unterstützt mehr als 30 Anbieter. Keine Programmierkenntnisse erforderlich. Funktioniert auch offline mit Ollama.
</TLDR>

## Schritt 1: Konfigurieren Sie Ihren ersten LLM-Anbieter (2 Min.)

### Option A: Cloud-Anbieter (OpenAI, Anthropic, usw.)

1. Öffnen Sie **Einstellungen** → **Notemd**
2. Unter **LLM Provider-Konfiguration** wählen Sie einen Anbieter aus dem Dropdown-Menü aus
3. Geben Sie Ihren **API Schlüssel** ein
4. (Optional) Klicken Sie auf **„Modellliste anfordern“**, um die verfügbaren Modelle anzusehen
5. Wählen Sie ein Modell für jede Aufgabentyp.

**Beliebte Auswahl:**
- **OpenAI**: `gpt-4o-mini` (schnell, günstig)
- **Anthropic**: `claude-sonnet-4-6` (hohe Qualität)
- **DeepSeek**: `deepseek-chat` (sehr günstig)

### Option B: Lokal Ollama (Kein API Schlüssel erforderlich)

1. Installieren Sie [Ollama](https://ollama.com/)
2. Modell herunterladen: `ollama pull llama3.1`
3. Wählen Sie in den Einstellungen von Notemd **„Ollama"** als Anbieter aus
4. Modellname: `llama3.1` (oder jedes andere heruntergeladene Modell)

## Schritt 2: Versuchen Sie Ihre erste Aufgabe (1 Min.)

### Wiki-Links zu einer Notiz hinzufügen

1. Öffnen Sie jede Notiz mit sinnvollem Inhalt
2. Rechtsklick im Editor
3. Wählen Sie **„Datei verarbeiten (Verknüpfungen hinzufügen)“** aus
4. Warten Sie ein paar Sekunden.
5. Die Schlüsselkonzepte sind jetzt `[[wiki-linked]]`!

**Beispiel:**

**Vorher:**
```
Machine learning models use neural networks to process data.
```

**Nach:**
```
[[Machine learning]] models use [[neural networks]] to process data.
```

## Schritt 3: Funktionen erkunden (2 Min.)

### Konzeptnotizen extrahieren

1. Rechtsklick → **„Konzepte aus der aktuellen Datei extrahieren“**
2. Notemd erstellt separate Dateien für jedes Konzept
3. Überprüfen Sie Ihren konfigurierten Konzeptordner (Standard: `concepts/`)
4. Jede Konzeptnotiz enthält einen Backlink zu Ihrer ursprünglichen Notiz

### Forschung und Zusammenfassung

1. Heben Sie ein Thema in Ihrer Notiz hervor
2. Rechtsklick → **„Forschung und Zusammenfassung“**
3. Notemd führt eine Websuche durch (Tavily oder DuckDuckGo)
4. Zusammenfassung fügt Ihrer Notiz zusammen mit Quellen hinzu

### Einst-Klick-Abläufe

1. Überprüfen Sie die **linke Seitenleiste** auf Schaltflächen für den Workflow
2. Klicken Sie auf **„One-Click Extract“** in jeder Notiz
3. Es läuft wie folgt: Links hinzufügen → Konzepte extrahieren → Recherche durchführen → Diagramm erstellen
4. Alles mit einem Klick!

## Häufige Erstaufgaben

### Aufgabe 1: Ein Papier/Artikel verarbeiten

```
1. Paste article content into a note
2. Run "Process file (add links)"
3. Run "Extract concepts"
4. Run "Research and summarize" on unfamiliar terms
5. Run "Summarize as Mermaid diagram" for visual structure
```

### Aufgabe 2: Notizen übersetzen

```
1. Highlight text to translate
2. Right-click → "Translate to target language"
3. Set target language in settings first
```

### Aufgabe 3: Erstellen Sie eine Wissenskarte

```
1. Process 5-10 related notes with "add links"
2. Run "Extract concepts" on each
3. Open Graph view to see connections
4. Your knowledge graph is forming!
```

## Tipps zur Anpassung

### Modellauswahl pro Aufgabe

Sie können unterschiedliche Modelle für verschiedene Aufgaben verwenden:

| Aufgabe | Empfohlenes Modell | Warum |
|------|------------------|-----|
| Links hinzufügen | `gpt-4o-mini`, `deepseek-chat` | Schnell, viele Anrufe |
| Konzepte extrahieren | `claude-sonnet-4-6`, `gpt-4o` | Qualitätsextraktion |
| Forschung | `perplexity`, `gpt-4o` | webbasiert |
| Übersetzung | `gemini-flash`, `gpt-4o` | Mehrsprachig |
| Diagramme | `claude-opus`, lokales Modell | Komplexe Logik |

Konfigurieren in: **Einstellungen → Notemd → modell-spezifische Aufgaben**

### Benutzerdefinierte Workflows

Erstellen Sie Ihre eigene Schaltfläche:

1. Einstellungen → Notemd → Arbeitsabläufe
2. Klicken Sie auf „Workflow hinzufügen“.
3. Geben Sie den DSL ein: `process-current-add-links>extract-concepts-current>research-and-summarize`
4. Nennen Sie es: „Mein Forschungsfluss“
5. Ein neuer Button erscheint im Seitenbereich

## Fehlerbehebung

### "API Schlüssel ungültig"
- Überprüfen, ob der Schlüssel korrekt ist (keine zusätzlichen Leerzeichen)
- Überprüfen, ob das Konto Überziehungskredite hat
- Testen Sie mit der Schaltfläche „Verbindungsprüfung“ in den Einstellungen

### „Modell nicht gefunden“
- Klicken Sie auf „Modellliste anfordern“, um die verfügbaren Modelle anzusehen.
- Einige Anbieter verlangen bestimmte Modellnamen (z. B. `gpt-4o` und nicht `gpt-4`).

### Links erscheinen nicht.
- Stellen Sie sicher, dass die Notiz ausreichend Inhalt enthält (>50 Wörter).
- Probieren Sie ein leistungsstärkeres Modell aus.
- Überprüfen, ob der Aufruf von LLM erfolgreich war (siehe Diagnosepanel).

### Langsame Leistung
- Verwenden Sie schnellere Modelle (`gpt-4o-mini` anstelle von `gpt-4o`)
- Größe des Chunk in den Einstellungen verringern
- Parallelverarbeitung für Batch-Aufgaben aktivieren
