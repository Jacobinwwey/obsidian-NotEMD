# Installation

<TLDR>
**Installieren Sie Notemd in weniger als 2 Minuten.** Zwei Methoden: (1) Obsidian Community Plugins (empfohlen, ein-Klick-Installation), oder (2) Manuelle Installation über GitHub Releases. Erfordert Obsidian 0.15.0+. Funktioniert unter Windows, macOS, Linux, iOS und Android. Keine externen Abhängigkeiten. Mehr als 4k Downloads, lizenziert nach MIT.
</TLDR>

## Methode 1: Community-Plugins (empfohlen)

1. Öffnen Sie die **Obsidian Einstellungen** (⚙️-Icon oder `Ctrl/Cmd + ,`)
2. Gehen Sie zu **Community Plugins**
3. Klicken Sie auf **Browse**
4. Suche nach **"Notemd"**
5. Klicken Sie auf **Installieren**
6. Klicken Sie auf **Enable**

**Erledigt!** Sie werden das Notemd-Icon in Ihrer linken Seitenleiste sehen.

## Methode 2: Manuelle Installation

### Voraussetzungen
- Obsidian 0.15.0 oder neuer
- Git (optional, zum Clonen)

### Schritte

1. **Dateien herunterladen** von [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest):
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. **Finden Sie den Plugin-Ordner Ihres Tresors:**
   ```
   <your-vault>/.obsidian/plugins/
   ```

Falls das Verzeichnis `plugins` nicht existiert, erstellen Sie es.

3. **Erstellen Sie den Ordner Notemd:**
   ```
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. **Kopieren Sie die drei Dateien** in den Ordner `notemd`.

5. **Neuladen von Obsidian:**
   - Drücken Sie `Ctrl/Cmd + R`, oder
   - Restarte Obsidian

6. **Plugin aktivieren:**
   - Einstellungen → Community-Plugins
   - Suche nach "Notemd" in der Liste
   - Schalte es ein.

## Alternative: Über BRAT installieren

Wenn Sie den [BRAT-Plugin](https://github.com/TfTHacker/obsidian42-brat) verwenden:

1. Öffne die Einstellungen von BRAT
2. Klicken Sie auf „Beta-Plugin hinzufügen“.
3. Eingeben: `Jacobinwwey/obsidian-NotEMD`
4. Klicken Sie auf „Plugin hinzufügen“.
5. Aktiviere Notemd in den Community-Plugins

## Überprüfung

Nach der Installation überprüfen Sie, ob Notemd funktioniert:

1. Öffne jede Notiz
2. Rechtsklick im Editor
3. Sie sollten Notemd Befehle wie folgt sehen:
   - „Datei verarbeiten (Verknüpfungen hinzufügen)“
   - „Konzepte extrahieren“
   - "Forschung durchführen und zusammenfassen"

## Deinstallation

Um Notemd zu entfernen:

1. Einstellungen → Community-Plugins
2. Finde „Notemd“
3. Klicken Sie auf das ❌-Icon
4. Optional können Sie das Plugin-Verzeichnis löschen: `<vault>/.obsidian/plugins/notemd/`

Ihre Daten (Notizen, Konzeptnotizen usw.) bleiben unberührt.

## Aktualisieren

### Automatische Aktualisierung (empfohlen)
Obsidian überprüft automatisch nach Updates für Plugins. Wenn eine neue Version verfügbar ist:
1. Einstellungen → Community-Plugins
2. Klicken Sie auf „Nach Updates suchen“.
3. Klicken Sie auf „Aktualisieren“ neben Notemd

### Manuelle Aktualisierung
Laden Sie die neueste Version herunter und ersetzen Sie die drei Dateien in Ihrem Plugin-Ordner.
