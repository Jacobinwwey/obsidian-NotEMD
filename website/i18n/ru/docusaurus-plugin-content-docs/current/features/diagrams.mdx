# Диаграммы

<TLDR>
Выберите **тип диаграммы**, который вам нужен, затем выберите её **редактируемый исходный формат**. Посмотрите превью результата в Obsidian и экспортируйте его в форматах SVG, PNG или PDF.
</TLDR>

## Что такое три возможных варианта

| Вариант | Что он контролирует | Примеры |
|---|---|---|
| Тип диаграммы | Значение, которое будет отображаться на диаграмме | Диаграмма потока, последовательная диаграмма, схема цепи, график данных |
| Исходный формат | Редактируемый или повторно используемый файл, который создает Notemd | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| Формат экспорта | Изображение или документ, предназначенный для обмена | SVG, PNG, PDF |

Draw.io и Drawnix относятся к **исходным форматам**, а не к типам диаграмм. CircuitikZ — это исходный формат, разработанный специально для схем цепей. Одно и то же превью можно экспортировать в несколько визуальных форматов, поэтому выбор CircuitikZ не ограничивает возможности экспорта только SVG.

## Выбор исходного формата

| Формат | Для чего лучше всего подходит | Файл исходника |
|---|---|---|
| Mermaid | Диаграммы, сохраняемые непосредственно в Markdown | Блок кода Mermaid |
| JSON Canvas | Работа с нативной платформой canvas в Obsidian | Файл с расширением `.canvas` |
| Draw.io | Дальнейшая редактура в программе diagrams.net или совместимых инструментах | Файл с расширением `.drawio` |
| Drawnix | Редактирование в стиле ручного наброска или на доске | Файл с расширением `.drawnix` |
| CircuitikZ | Структурированные электрические и транзисторные схемы | Файл с расширением `.tex` |

Если вам не нужен конкретный редактор, оставьте поле **Предпочтительный исходный формат** в значении **Auto**. Notemd сам выберет подходящий формат для выбранного типа диаграммы.

## Создание и экспорт диаграммы

1. Откройте заметку или выберите текст, который необходимо визуализировать.
2. В боковой панели Notemd или в палитре команд выберите опцию **Создать диаграмму**.
3. Выберите **тип диаграммы**, либо оставьте значение на **Автодетекция**.
4. Выберите **исходный формат**, либо оставьте его на **Auto**.
5. Просмотрите полученное превью.
6. Выберите один из вариантов: **Сохранить файл исходника**, **Экспортировать как SVG**, **Экспортировать как PNG** или **Экспортировать как PDF**.

Для форматов PNG и PDF используется значение PPI, заданное в настройках экспорта изображений в Notemd. SVG остается векторным изображением, поэтому на него не влияет значение PPI.

## Схемы цепей

Для создания поддерживаемых схем цепей выберите **Схема цепи** в сочетании с **файлом исходника CircuitikZ**. Notemd сначала генерирует структурированный описатель схемы, а затем записывает код в формате CircuitikZ. Такой подход надежнее, чем просить модель генерировать неограниченный LaTeX-код.

Текущая поддержка CircuitikZ преднамеренно ограничена. Лучше всего работают схемы на основе общедоступных усилителей NMOS и поддерживаемых блоков CMOS. Всегда проверяйте превью и исходный код перед использованием генерированной схемы в технических или публикационных целях.

Превью в формате SVG создается Notemd для удобства проверки. Оно не является доказательством того, что локальная установка LaTeX или TikZJax успешно скомпилировала файл с расширением `.tex`.

## Устранение неполадок

### Результат — неверный тип диаграммы

Вместо использования режима автодетекции явно укажите нужный тип диаграммы. Для схем цепей выберите опцию **Схема цепи** и воспользуйтесь командой **Создать диаграмму**, а не устаревшей командой **Суммировать как диаграмму Mermaid**.

### Исходный формат не подходит

Верните значение исходного формата в **Auto**, либо выберите формат, предназначенный для конкретного типа диаграммы. CircuitikZ предназначен только для схем цепей; Draw.io и Drawnix подходят для определенных структурированных диаграмм.

### Отсутствуют кнопки экспорта

Для форматов SVG, PNG и PDF требуется визуальное превью. Если доступен только текстовый исходник, сохраните файл исходника и откройте его в соответствующем внешнем инструменте.

### Файл CircuitikZ не компилируется во внешних средах

Убедитесь, что ваша среда LaTeX включает пакет `circuitikz`. Плагин не включает полную сборку LaTeX.
