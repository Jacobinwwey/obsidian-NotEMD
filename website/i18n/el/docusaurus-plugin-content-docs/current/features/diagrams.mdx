# Diagramas

<TLDR>
Elige **qué tipo de diagrama necesitas** y luego selecciona su **formato de origen editable**. Previsualiza el resultado en Obsidian y guárdalo como SVG, PNG o PDF.
</TLDR>

## Comprender las tres opciones

| Opción | Qué controla | Ejemplos |
|---|---|---|
| Tipo de diagrama | El significado del diagrama | Diagrama de flujo, diagrama de secuencia, diagrama eléctrico, gráfico de datos |
| Formato de origen | El archivo editable o reutilizable que crea Notemd | Mermaid, JSON Canvas, Draw.io, Drawnix, CircuitikZ |
| Formato de exportación | La imagen o documento utilizado para compartir | SVG, PNG, PDF |

Draw.io y Drawnix son **formatos de origen**, no tipos de diagrama. CircuitikZ es un formato de origen diseñado específicamente para diagramas eléctricos. La misma previsualización puede exportarse a varios formatos visuales, por lo que seleccionar CircuitikZ no te limita únicamente a SVG.

## Elegir un formato de origen

| Formato | Mejor para | Archivo de origen |
|---|---|---|
| Mermaid | Diagramas que se mantienen directamente en Markdown | Bloque de código Mermaid |
| JSON Canvas | Flujos de trabajo nativos en el canvas de Obsidian | `.canvas` |
| Draw.io | Edición posterior en diagrams.net o herramientas compatibles | `.drawio` |
| Drawnix | Edición al estilo de dibujo a mano o pizarra | `.drawnix` |
| CircuitikZ | Circuitos eléctricos y de transistores estructurados | `.tex` |

Deja **Formato de origen preferido** en **Automático** si no necesitas un editor específico. Notemd elegirá un formato compatible según el tipo de diagrama seleccionado.

## Generar y exportar un diagrama

1. Abre la nota o selecciona el texto que deseas visualizar.
2. Elige **Generar diagrama** desde la barra lateral de Notemd o el panel de comandos.
3. Selecciona un **tipo de diagrama**, o déjalo en **Detectar automáticamente**.
4. Elige un **formato de origen**, o déjalo en **Automático**.
5. Revisa la previsualización.
6. Elige **Guardar archivo de origen**, **Exportar SVG**, **Exportar PNG** o **Exportar PDF**.

PNG y PDF utilizan la resolución PPI configurada para la exportación de imágenes en los ajustes de Notemd. SVG sigue siendo una imagen vectorial y no se ve afectado por el valor de PPI.

## Diagramas eléctricos

Para solicitudes de diagramas eléctricos compatibles, elige **Diagrama eléctrico** con **Archivo de origen CircuitikZ**. Notemd primero genera una descripción estructurada del circuito y luego escribe el código de origen en formato CircuitikZ. Este método es más fiable que solicitar al modelo que genere código LaTeX sin restricciones.

La compatibilidad actual con CircuitikZ está intencionalmente limitada. Los amplificadores NMOS de fuente común y los componentes CMOS soportados funcionan mejor. Siempre revisa la previsualización y el código de origen antes de utilizar un circuito generado en trabajos técnicos o de publicación.

La previsualización en SVG es una herramienta de revisión generada por Notemd. No constituye prueba de que una instalación local de LaTeX o TikZJax haya compilado con éxito el archivo `.tex`.

## Solución de problemas

### El resultado es del tipo de diagrama incorrecto

Selecciona explícitamente el tipo de diagrama en lugar de usar la opción de detección automática. Para diagramas eléctricos, elige **Diagrama eléctrico** y utiliza el comando **Generar diagrama**, en lugar del comando obsoleto **Resumir como diagrama Mermaid**.

### Un formato de origen no es adecuado

Vuelve a establecer el formato de origen en **Automático**, o elige un formato diseñado específicamente para ese tipo de diagrama. CircuitikZ está destinado a diagramas eléctricos; Draw.io y Drawnix están pensados para diagramas estructurados compatibles.

### Faltan los botones de exportación

SVG, PNG y PDF requieren una previsualización visual. Si solo está disponible el texto de origen, guarda el archivo de origen y ábrelo en la herramienta externa correspondiente.

### Un archivo CircuitikZ no se compila en otro entorno

Verifica que tu entorno LaTeX incluya el paquete `circuitikz`. El plugin no incluye una distribución completa de LaTeX.
