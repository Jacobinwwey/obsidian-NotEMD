# 설치

<TLDR>
**2분 이내에 Notemd를 설치하세요.** 두 가지 방법이 있습니다: (1) Obsidian 커뮤니티 플러그인(권장, 원클릭 설치), 또는 (2) GitHub Releases를 통한 수동 설치. Obsidian 0.15.0+가 필요합니다. Windows, macOS, Linux, iOS, Android에서 작동합니다. 외부 의존성이 없습니다. 4k회 이상 다운로드되었으며 MIT 라이선스를 따릅니다.
</TLDR>

## 방법 1: 커뮤니티 플러그인 (권장)

1. **Obsidian 설정**을 엽니다(⚙️ 아이콘 또는 `Ctrl/Cmd + ,`).
2. **Community plugins**로 이동하세요.
3. **찾아보기**를 클릭하세요
4. **"Notemd"**를 검색하세요.
5. **Install**을 클릭하세요.
6. **활성화**를 클릭하세요.

**완료되었습니다!** 왼쪽 사이드바에 Notemd 아이콘이 표시될 것입니다.

## 방법 2: 수동 설치

### 사전 요구 사항
- Obsidian 0.15.0 이상
- Git (클론용, 선택 사항)

### 단계들

1. [GitHub Releases](https://github.com/Jacobinwwey/obsidian-NotEMD/releases/latest)에서 **파일 다운로드**:
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. **귀하의 볼트의 플러그인 폴더를 찾으세요:**
   ```
   <your-vault>/.obsidian/plugins/
   ```

`plugins` 폴더가 존재하지 않으면 생성하세요.

3. **Notemd 폴더 생성:**
   ```
   <your-vault>/.obsidian/plugins/notemd/
   ```

4. **세 개의 파일을** `notemd` 폴더에 복사하세요.

5. **Obsidian 재로드:**
   - `Ctrl/Cmd + R`를 누르거나,
   - Obsidian를 재시작하세요.

6. **플러그인 활성화하기:**
   - 설정 → 커뮤니티 플러그인
   - 목록에서 "Notemd"를 찾아보세요.
   - 켜기로 전환하세요.

## 대안: BRAT를 통해 설치하기

[BRAT 플러그인](https://github.com/TfTHacker/obsidian42-brat)을 사용하는 경우:

1. BRAT 설정을 엽니다.
2. “Beta 플러그인 추가”를 클릭하세요.
3. 입력: `Jacobinwwey/obsidian-NotEMD`
4. “플러그인 추가”를 클릭하세요.
5. Community 플러그인에서 Notemd를 활성화하세요

## 검증

설치가 완료된 후 Notemd이 제대로 작동하는지 확인하세요:

1. 어떤 노트든 열기
2. 편집기에서 마우스 오른쪽 버튼 클릭
3. Notemd와 같은 명령들을 볼 수 있습니다:
   - "파일 처리 (링크 추가)"
   - "개념 추출"
   - "조사하고 요약하기"

## 제거

Notemd를 제거하려면:

1. 설정 → 커뮤니티 플러그인
2. Notemd를 찾아주세요.
3. ❌ 아이콘을 클릭하세요.
4. 선택적으로 플러그인 폴더를 삭제할 수 있습니다: `<vault>/.obsidian/plugins/notemd/`

귀하의 데이터(노트, 컨셉 노트 등)는 그대로 유지됩니다.

## 업데이트 중입니다

### 자동 업데이트 (권장)
Obsidian는 자동으로 플러그인 업데이트를 확인합니다. 새 버전이 준비되면:
1. 설정 → 커뮤니티 플러그인
2. “업데이트 확인”을 클릭하세요.
3. Notemd 옆에 있는 “Update”를 클릭하세요.

### 수동 업데이트
최신 버전을 다운로드한 후 플러그인 폴더에 있는 세 개의 파일을 교체하세요.
